# 个人知识库

这是我的个人知识库，用于存储、整理和关联各类知识与信息。

## 文件夹结构

| 文件夹 | 用途 |
|--------|------|
| `Inbox` | 收件箱 - 临时存放待整理的内容 |
| `Daily` | 每日笔记 - 记录每日想法和日志 |
| `Projects` | 项目笔记 - 各类项目相关文档 |
| `Areas` | 关注领域 - 长期关注的主题 |
| `Resources` | 资源库 - 收藏的参考资料 |
| `Archives` | 归档 - 已完成或过时的内容 |
| `Templates` | 模板 - 可复用的笔记模板 |

## 使用说明

### AI 助手工作流
1. **新内容**：发送给配置好的 AI 助手（加载 `system-prompt.md`）
2. **自动处理**：AI 助手自动完成整理、格式化、分类存储
3. **关联增强**：自动建立双向链接和标签系统
4. **结果确认**：AI 助手返回存储位置和关联信息

### 系统提示词
- `system-prompt.md` - AI 助手的系统提示词配置
- 将此文件内容作为 AI 助手的 system prompt 使用
