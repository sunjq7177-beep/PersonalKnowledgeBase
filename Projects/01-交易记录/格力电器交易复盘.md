# 格力电器交易复盘

## 先看结论

| 项目 | 结论 |
|------|------|
| 这笔钱 | 纯低估值慢修复的钱，外加一点点业绩改善预期 |
| 结果 | ❌ 失败案例，最终因业绩不及预期和机会成本退出 |
| 样本性质 | 反样本 |
| 是否适合复制 | 不适合作为主路径 |
| 一句话复盘 | 便宜不等于会上涨，没有明确催化却给了高仓位，最后只能靠时间价值压力离场 |

## 交易事实

| 维度 | 内容 |
|------|------|
| 买入时间 | 2025 年 3 月左右 |
| 卖出时间 / 当前状态 | 2025 年 8 月左右卖出，当前已空仓 |
| 仓位 | 约 50% |
| 核心变量 | 低估值、高分红、相对同行折价 |
| 验证路径 | 半年报经营稳态、未来出海改善、估值修复催化 |

## 当时为什么买

- 认为格力估值大约 7 倍 PE，明显低于美的、海尔。
- 觉得公司利润率和主业质量不差，高分红提供安全边际。
- 额外附加了一层“未来可能出海改善”的中期想象。

## 过程与关键转折

1. 采取分笔加仓，最终把仓位打到约 50%。
2. 持有期间 2025 年半年报明显低于预期，“稳经营”这层逻辑被破坏。
3. 没有在失效点第一时间退出，而是拖到机会成本压力越来越大时，才切换到别的方向。

## 复盘判断

### 做对了什么

- 发现逻辑失效后，最终没有死守。
- 能看见低估值背后的安全边际，这一点本身不是错。

### 做错了什么

- 把“便宜”误当成上涨驱动力。
- 缺少明确催化和验证路径，却给了很高仓位。
- 卖出不是按预设规则，而是被时间价值和机会成本逼出来的。

### 以后怎么处理同类机会

- 没有明确重估催化的静态低估值，不再作为主仓机会。
- 这类交易如果再做，只能在催化和验证路径足够清楚时参与。
- “便宜”以后只能是辅助条件，不能再做主理由。
