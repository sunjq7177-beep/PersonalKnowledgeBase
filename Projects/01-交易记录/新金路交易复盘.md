# 新金路交易复盘

## 先看结论

| 项目 | 结论 |
|------|------|
| 这笔钱 | 题材情绪 + 技术试错的钱 |
| 结果 | ❌ 两次试错后按纪律止损，整体亏损约十几个点 |
| 样本性质 | 试错样本 |
| 是否适合复制 | 只适合小仓位试错，不适合升级为主逻辑 |
| 一句话复盘 | 这笔亏损本身不是问题，真正重要的是它证明了自己可以小仓位试错，但绝不能把题材情绪交易上升成主战场 |

## 交易事实

| 维度 | 内容 |
|------|------|
| 买入时间 | 2026 年 2 月左右 |
| 卖出时间 / 当前状态 | 两次买入后均止损离场 |
| 仓位 | 约 5% 到 10% |
| 核心变量 | 矿业转型题材、板块情绪、技术形态 |
| 验证路径 | 板块强弱、异常波动、公司项目推进、技术止损位 |

## 当时为什么买

- 看的是矿业资产、复工和技改带来的题材预期。
- 公司有资源概念，市场也有资金运作和题材联想。
- 没有直接追消息，而是等技术形态出现后才小仓位介入。

## 过程与关键转折

1. 买入前已经知道这是高噪音、高情绪的交易，所以仓位控制得比较轻。
2. 买卖主要都按技术执行，而不是按基本面持有。
3. 随着有色板块情绪退潮和资金博弈加剧，两次都触发止损离场。

## 复盘判断

### 做对了什么

- 小仓位试错，风险暴露克制。
- 两次试错都按纪律止损，没有因为题材叙事去死扛。
- 说明自己在这类机会里，至少已经能做到“动作比故事重要”。

### 做错了什么

- 主逻辑基本面含量不足，更多是在交易故事和情绪。
- 买入时点已经接近情绪活跃区，不是低位埋伏。
- 把“庄股 / 资金运作”之类噪音信息放进观察范围，会污染系统。

### 以后怎么处理同类机会

- 只允许小仓位、纯技术、快止损。
- 这类机会以后不进入主仓候选池。
- 如果连题材强度和情绪阶段都说不清，直接放弃。
