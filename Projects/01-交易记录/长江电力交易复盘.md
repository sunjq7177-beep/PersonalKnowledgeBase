# 长江电力交易复盘

## 先看结论

| 项目 | 结论 |
|------|------|
| 这笔钱 | 高股息 / 类债配置的钱 |
| 结果 | ⚠️ 大致平出，可能略亏 |
| 样本性质 | 反样本 |
| 是否适合复制 | 不适合作为主路径 |
| 一句话复盘 | 不是资产有问题，而是这种低弹性、慢兑现的收益方式和自己的真实偏好不匹配 |

## 交易事实

| 维度 | 内容 |
|------|------|
| 买入时间 | 2025 年 4 月左右 |
| 卖出时间 / 当前状态 | 中途减仓并最终清仓 |
| 仓位 | 约 30% |
| 核心变量 | 高股息、低波动、长期稳定现金流 |
| 验证路径 | 分红、资产质量、经营稳态、政策和治理扰动 |

## 当时为什么买

- 把它视为“类债券”的股票资产，希望拿稳定回报。
- 看重高股息、合理估值和长期经营稳定性。
- 认为折旧下降后，公司利润释放仍有确定性。

## 过程与关键转折

1. 以替代债券配置的思路进场，给了较高仓位。
2. 年报本身不差，但后续资金占用和地方政策扰动强化了对国企治理外部性的担忧。
3. 虽然并未出现明显经营恶化，但主观持有体验变差，最终逐步减仓并清仓。

## 复盘判断

### 做对了什么

- 识别出了资产的稳健属性。
- 在感到治理和持有体验不匹配后，最终选择退出。

### 做错了什么

- 把配置型逻辑和交易型思维混在一起。
- 30% 仓位过重，但这笔交易本来又不是高弹性赚钱模式。
- 对国企治理外部性和持有扰动前置不够。

### 以后怎么处理同类机会

- 不再把高股息 / 类债配置作为主要赚钱路径。
- 如果以后出于资产配置需要参与，也要和“交易主路径”彻底分开看待。
- 这类资产以后不能再用主仓交易心态去要求它提供资本利得。
