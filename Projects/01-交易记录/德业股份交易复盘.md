# 德业股份交易复盘

## 先看结论

| 项目 | 结论 |
|------|------|
| 这笔钱 | 成长预期差的钱，后面叠加了估值抬升和趋势强化 |
| 结果 | ✅ 成功样本，整体盈利较高 |
| 样本性质 | 正样本 |
| 是否适合复制 | 适合复制，但仓位表达要收敛 |
| 一句话复盘 | 这是最接近主路径的一笔，问题不在方向，而在于仓位过重和后半段已不完全知道自己在挣哪类钱 |

## 交易事实

| 维度 | 内容 |
|------|------|
| 买入时间 | 2025 年 8 月左右 |
| 卖出时间 / 当前状态 | 持有至阶段高点后卖出 |
| 仓位 | 最高约 60% |
| 核心变量 | 海外渠道优势、储能景气、盈利质量 |
| 验证路径 | 财报、渠道表现、行业景气、市场对估值的再定价 |

## 当时为什么买

- 看好公司在巴西、欧洲等海外市场的渠道和产品竞争力。
- 认为储能景气向上，公司 ROE 和盈利质量较好。
- 买入时估值不算贵，希望拿到“景气上行 + 估值抬升”的双重收益。

## 过程与关键转折

1. 逐步建仓并持续加仓，最终形成主仓。
2. 行业景气、板块走势和风险偏好整体向上，公司即使短期业绩 miss，市场也很快修复。
3. 到后半段时，股价上涨已经部分超出最初基本面理解，开始混入趋势和情绪强化，但整体仍顺利兑现。

## 复盘判断

### 做对了什么

- 切入时点较早，属于“市场尚未完全拥挤前”。
- 逻辑有产业支撑，也有公司边际优势，符合主路径特征。
- 这笔交易验证了自己确实更适合做“变化已经开始，但还没完全一致”的机会。

### 做错了什么

- 最高仓位约 60%，风险暴露过重。
- 到后半段时，没有及时意识到自己已经部分在挣趋势和情绪的钱。
- 仍然会被横向比较和机会成本影响持有体验。

### 以后怎么处理同类机会

- 同类机会继续做，而且应作为主路径样本。
- 但以后 A 类机会也不再用极端重仓去表达。
- 当走势明显脱离最初基本面理解时，要重新定义自己在挣什么钱，不能稀里糊涂拿到底。
