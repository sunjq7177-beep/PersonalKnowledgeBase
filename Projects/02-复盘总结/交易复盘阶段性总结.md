# 交易复盘阶段性总结

## 当前总判断

> **最适合挣的是“有基本面锚点的变化与修正的钱”；最不适合挣的是“纯防守的钱、最拥挤后半段的钱、以及纯消息情绪的钱”。**

## 现在到底适合做什么

### 三类主路径

| 类型 | 为什么适合 | 代表样本 |
|------|------------|----------|
| 成长预期差 | 有基本面支撑，也有价格弹性 | [[德业股份交易复盘]] |
| 困境 / 政策修复预期差 | 市场过度悲观，但修复已开始 | [[天立国际控股交易复盘]] |
| 高质量龙头业绩验证与重估 | 兑现更稳，持有体验更好 | [[腾讯控股交易复盘]] |

## 现在明确不适合做什么

| 类型 | 为什么不适合 | 代表样本 |
|------|--------------|----------|
| 纯低估值慢修复 | 催化弱，时间成本太高 | [[格力电器交易复盘]] |
| 高股息 / 类债配置 | 低弹性，收益诉求不匹配 | [[长江电力交易复盘]] |
| 拥挤成长后半段 | 对纪律和估值承压要求太高 | [[阳光电源交易复盘]] |
| 纯题材 / 纯情绪接力 | 噪音高，不是长期优势区 | [[新金路交易复盘]] |

## 七笔样本到底说明了什么

| 标的 | 样本性质 | 这笔交易真正说明的事 |
|------|----------|------------------------|
| [[阳光电源交易复盘]] | 反样本 | 不是看不懂成长，而是不适合在高拥挤阶段用混合体系重仓承受高波动 |
| [[格力电器交易复盘]] | 反样本 | 不是不能看低估值，而是不能把“便宜”误当成催化 |
| [[德业股份交易复盘]] | 正样本 | 最适合的是“变化已经开始、市场尚未完全一致”的成长预期差 |
| [[天立国际控股交易复盘]] | 正样本 | 也适合做“市场太悲观但修复已出现”的修复波段 |
| [[长江电力交易复盘]] | 反样本 | 纯防守型收益方式和真实偏好不匹配 |
| [[新金路交易复盘]] | 试错样本 | 可以小仓位试错，但题材情绪永远不升格成主逻辑 |
| [[腾讯控股交易复盘]] | 正样本 | 不只适合中小票预期差，也适合大市值高质量龙头重估 |

## 当前最核心的四个共性问题

### 1. 仓位表达和交易质量不匹配

- 差样本敢重仓。
- 好样本反而轻仓。

### 2. 决策语言经常切换

- 买入时一套逻辑。
- 持有时另一套逻辑。
- 卖出时再换一套逻辑。

### 3. 容易把“知道为什么买”误当成“适合长期复制”

- 有逻辑，不等于属于自己。
- 典型反例就是格力和长江电力。

### 4. 容易被热度、横向比较和噪音入口带偏

- 会被别的票涨得更快影响持有体验。
- 会让“听消息”“庄股联想”之类信息进入系统。

## 现在最重要的操作原则

1. 以后每笔交易先定义赚钱类型。
2. 每笔交易只抓一个主变量，最多一个辅助变量。
3. 仓位只跟机会质量挂钩，不跟主观兴奋度挂钩。
4. 技术面只负责节奏和风控，不负责改变交易类型。
5. 卖出必须提前写好三类触发：逻辑失效、节奏走坏、兑现完成。

## 后面 90 天怎么做

### 第一步：30 天内先把系统收窄

- 只做三类主路径。
- 所有“别人说的 / 看着像机会的”先排除。

### 第二步：30 到 60 天把交易卡片固定

- 每笔交易都要有计划卡片。
- 盘中不再临时找理由。

### 第三步：60 到 90 天只看结构数据

- A 类交易胜率
- A 类平均盈亏比
- B 类是否拖累账户
- 最大亏损是否偏离主路径
- 情绪型卖出占比

## 当前给自己的定位

> **研究驱动的中期预期差交易者**

不是纯价值，也不是纯短线。核心是先找到真实变化，再判断市场有没有定错价，最后用纪律把逻辑变成交易。

## 一句话收束

> **先把系统做窄、做稳、做干净，再谈稳定盈利。**
