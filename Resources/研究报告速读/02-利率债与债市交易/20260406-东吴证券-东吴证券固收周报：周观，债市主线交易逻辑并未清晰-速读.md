# 东吴证券固收周报：债市主线交易逻辑并未清晰

## 一句话结论

债券市场在"滞"与"胀"的逻辑中反复博弈，主线交易逻辑尚不清晰，10年期国债收益率维持窄幅震荡，海外冲突的冲击正在钝化。

## 最重要的边际变化

1. **美伊冲突升级**：特朗普宣布未来2-3周将对伊朗进行"极其猛烈打击"，未如市场预期般缓和，通胀预期持续发酵
2. **制造业PMI重返荣枯线上**：3月PMI 50.4%，生产指数+1.8pp、新订单指数+3pp、新出口订单+4.1pp，外需保持强劲
3. **美国PMI结构分化**：制造业扩张（52.3%）但服务业收缩（49.8%），反映库存周期改善但消费放缓
4. **美联储降息预期后移**：4月/6月维持利率不变概率分别为99.5%/96.7%，12月仍有69.5%概率维持当前利率
5. **央行政策定力**：强调"降低融资中间费用"而非贸然降息，流动性维持宽松但缺乏进一步刺激

## 未来可能出现的现象

1. **3月PPI或将转正**：高油价推动下，3月PPI受美伊冲突影响可能转正，但传导至CPI存在时滞
2. **供给缺口或在4月底显现**：中东产油国油罐使用率攀升、产量下行，全球供给缺口或集中于4月底显现
3. **海峡通航修复斜率较缓**：虽逐步有条件放行，但保费高企，绕行好望角确定性更强
4. **美伊冲突三方打击范围持续扩大**：形势仍然多变，高度不确定性延续
5. **下周物价数据**：建议关注CPI/PPI公布情况

## 投资含义

1. **债券市场**：维持10年期国债收益率上半年1.9%顶部判断，需看到持续且由内需驱动的物价回升才会出现明显趋势
2. **海外资产配置**：建议构建对冲组合，黄金与美债修复空间仍足，HALO标的优先级高于科技成长
3. **城投债**：提前兑付持续推进，本周山西省兑付0.80亿元
4. **流动性**：央行维持精细化管理导向，宽松流动性延续但缺乏进一步降息预期

## 作者主观观点压缩版

本周债券市场主线逻辑不清晰，对海外冲突反应钝化，在"滞"和"胀"的逻辑中反复博弈。国内PMI重返荣枯线但主要反映季节性规律，价格指数回升或挤压企业利润。外部美伊冲突高度不确定性延续，但PPI向CPI传导路径不直接。央行着力精细化管理而非贸然降息，流动性平稳但缺乏催化剂。综合来看，债券市场需要看到持续且由内需驱动的物价回升，才会形成明确趋势。美联储维持"更高利率更久"路径，年内降息时点进一步后移至2026年下半年概率增大。
