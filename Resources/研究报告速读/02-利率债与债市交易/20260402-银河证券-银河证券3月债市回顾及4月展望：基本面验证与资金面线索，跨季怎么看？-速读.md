# 银河证券3月债市回顾及4月展望：基本面验证与资金面线索，跨季怎么看？

## 一句话结论

3月债市"长上短下"格局延续，10Y国债收益率回升至1.82%，期限利差走扩至60BP；4月核心关注一季度基本面数据成色、PPI转正后通胀再定价、央行态度边际收紧信号以及超长期特别国债落地带来的短期扰动。

## 核心数据卡

|------|------|------|

## 最重要的边际变化

1. **PMI重返扩张区间**：3月制造业PMI录得50.4%，较上月大幅回升1.4pct，供需价格同步回暖（原材料购进价格指数+9.1pct至63.9%）
2. **PPI即将转正**：测算3月PPI同比可能回升至1%以上，原油价格同比涨幅39.36%是核心驱动因素
3. **央行边际收紧信号出现**：4月1-2日仅分别投放5亿元地量7天逆回购，创近十年最低水平，调控意图显现
4. **配置盘与交易盘接力回暖**：配置盘增持由+2448亿升至+4839亿（保险发力），公募由净卖出转净买入+1090亿

## 未来可能出现的现象

1. **PPI同比转正**：3月PPI预计升至1%以上，输入性通胀从预期到实质落地
2. **CPI温和回落但仍正增**：春节效应消退后CPI增速下滑，但原油价格形成新的支撑
3. **资金面波动加大**：央行通过预期管理引导短端利率抬升，同业存单与政策利率倒挂风险释放
4. **超长期特国发行脉冲扰动**：参考2025年经验，公布后一周内DR007上行约3BP
5. **PMI扩张力度趋缓**：4月低基数优势减弱，预计PMI维持在扩张区间但斜率放缓
6. **地产小阳春分化持续**：一线城市成交回暖，但全国供需两端整体仍弱

## 投资含义

- **长端策略**：十债1.82%已回升至1.8%以上央行合意区间，"上有顶下有底"格局下关注调整带来的配置机会，建议适当进场把握窗口，交易盘顺势而为注意止盈
- **短端策略**：短端以政策利率1.4%为下限，当前1.22%附近博取赔率有限，关注回升风险
- **利差策略**：期限利差扩张至60BP，若PMI+PPI同步修复可能触发债市偏空定价（如2016年8月情形）
- **供给冲击**：超长端短期脉冲扰动可控，但需关注发行计划细节

## 作者主观观点压缩版

报告核心判断：**4月债市扰动大于趋势，震荡格局延续**。

分析师认为当前债市实质利空要素暂未出现但扰动要素较多。资金面短期易松难紧但央行已有边际收紧意图，需警惕倒挂收窄带来的波动。基本面方面关注一季度数据真实成色——若PMI与PPI同步修复，债市可能开始交易实质性输入性通胀风险。十债当前位置已不具备太大下行空间，交易情绪脆弱下收益率易上难下。策略上建议配置盘关注调整机会，交易盘顺势而为及时止盈。超长期特别国债落地带来的供给冲击短期可控，但需跟踪财政部发行公告与央行配合操作。
