# 阳光电源（300274.SZ）2025年报点评：年底业绩波动不改长期趋势，经营现金流同比大增

## 一句话结论

四季度单季盈利同比大幅下滑（归母净利润-54%），但全年看储能业务高增（+49%）驱动利润增长22%，经营现金流同比大增40%，维持"买入"评级。

**Q4单季**：
- 营收227.82亿元（-18.37%）
- 归母净利润15.80亿元（-54.02%）

## 最重要的边际变化

1. **储能成最强引擎**：储能收入增速49%远超逆变器7%，成为核心增长极
2. **现金流大幅改善**：经营净现金流169亿（+40%），现金净增42.8亿（+195%），回款效率显著提升
3. **四季度费用率跳升**：Q4销售/管理/研发费用率同比+2.17/+0.55/+1.72pct，侵蚀利润
4. **盈利能力微升**：毛利率+1.89pct至31.83%，净利率+0.70pct至15.17%
5. **新能源开发业务深度调整**：收入降21%，行业进入深度调整期

## 未来可能出现的现象

1. **中东冲突加速全球能源转型**：美伊冲突导致欧洲天然气价格单月飙升近90%，能源独立成各国共识，海外储能景气度有望继续上行
2. **储能业务占比将超50%**：按当前增速，储能收入有望在2026年超过逆变器成为第一大业务
3. **估值切换至2026年**：当前PE 26倍，2026E PE 18.56倍，估值具备吸引力
4. **回款压力持续改善**：经营现金流2026年预计因季节性回落至74亿，但整体改善趋势确立

## 投资含义

1. **短期波动不改长期逻辑**：Q4盈利下滑主要受季节性和费用集中确认影响，不影响储能高景气的核心逻辑
2. **储能是胜负手**：全球发货量增53%，海外市场贡献主要增量，欧洲能源危机将进一步刺激需求
3. **现金流质量改善是关键积极信号**：经营现金流增速（+40%）快于净利润增速（+22%），说明盈利质量提升
4. **目标估值**：2026年PE 18.56倍，维持"买入"评级，预计有15%以上上涨空间

## 作者主观观点压缩版

分析师对阳光电源维持乐观态度，认为：①储能业务是核心看点，49%的增速远超行业平均，欧洲能源危机将进一步拉动海外需求；②现金流大幅改善反映公司回款能力和经营质量提升；③短期费用率上升是扩张期正常现象，不影响长期盈利能力；④估值切换后具备安全边际，2026-2028年利润复合增速约20%，给予"买入"评级。
