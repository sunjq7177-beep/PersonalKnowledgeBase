# 阳光电源（300274.SZ）：四季度业绩承压，储能高景气持续

## 一句话结论

2025年全年归母净利润134.61亿元（+22%），但Q4净利润同比暴跌54%至15.80亿元，受低毛利项目集中交付拖累；储能业务维持高景气（出货+54%），公司下调2026-2027盈利预测并新增2028年预测，维持“买入”评级。

## 最重要的边际变化

1. **Q4业绩断崖下滑**：Q4营收227.82亿元（同比-18.37%），归母净利润15.80亿元（同比-54.02%），主要因低毛利新能源开发项目集中交付、碳酸锂涨价传导滞后、区域结构变化

2. **盈利预测大幅下调**：2026年归母净利润预测从200.75亿元下调至154.73亿元（-22.9%），2027年从222.46亿元下调至193.47亿元（-13.0%）

3. **储能业务结构性收缩**：国内发货仅7GWh（主动收缩低毛利市场），海外发货36GWh（同比+90%），成为核心增长引擎

4. **光伏逆变器毛利率提升**：毛利率34.66%，同比+3.76pct，但出货量同比略降

5. **AIDC新业务启动**：2025年开始布局固态变压器SST，2026年目标小批量交付，2027年进入批量供货阶段

## 未来可能出现的现象

1. **储能出货持续高增**：2026年全球储能市场预计保持30-50%高增速，公司储能出货目标超60GWh
2. **AIDC业务逐步兑现**：2026年下半年固态变压器产品有望落地小批量交付，2027年进入批量供货阶段
3. **毛利率压力**：2026年预测毛利率29.4%（较2025年下降2.4pct），主要因新能源开发业务拖累
4. **海外占比持续提升**：公司主动收缩国内低毛利市场，海外发货占比已达84%

## 投资含义

1. **短期承压不改长期逻辑**：Q4业绩下滑主要来自项目交付节奏和成本传导滞后，储能核心业务保持高景气
2. **估值具备安全边际**：当前PE 18x对应2026年预测，2027-2028年有望降至14.4x/12.1x，EV/EBITDA 12.8x处于历史低位
3. **储能出海是核心看点**：海外发货同比+90%，毛利率稳定36.49%，海外收入占比提升有望改善整体盈利能力
4. **新业务打开想象空间**：AIDC电源业务依托电力电子技术积累和800V平台优势，2027年后有望成为第二增长曲线

## 作者主观观点压缩版

开源证券维持“买入”评级，主要逻辑：
- 四季度业绩承压源于阶段性因素，非基本面恶化
- 储能业务量利齐升（出货+54%，毛利36.49%），海外高增长对冲国内收缩
- AIDC新赛道布局领先，2026年下半年逐步验证
- 盈利预测下调已充分反映短期压力，当前估值具备吸引力
