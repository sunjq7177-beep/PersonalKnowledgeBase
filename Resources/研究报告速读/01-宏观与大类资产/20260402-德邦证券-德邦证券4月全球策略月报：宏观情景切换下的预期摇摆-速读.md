# 德邦证券4月全球策略月报：宏观情景切换下的预期摇摆

## 一句话结论

3月中东局势推动市场主线从复苏逻辑向滞胀情景切换，4月市场预期将在通胀与滞胀之间来回摇摆，大类资产价格短期以避险为主导，但长期产业趋势仍值得博弈。

## 核心数据卡

| 指标 | 数值 | 备注 |
|------|------|------|
| 2月非农就业 | -9.2万 | 远低于预期 |
| 失业率 | 4.4% | 小幅上升 |
| CPI同比 | 2.4% | 符合预期 |
| 核心CPI同比 | 2.5% | 温和 |
| 工业增加值(1-2月) | +6.3% | 好于预期 |
| 进出口总额(1-2月) | +18.3% | 亮眼 |
| 降息预期时间 | 延后至2027年9月 | CME模型 |

## 最重要的边际变化

1. **市场主线切换**：从"中美传统行业复苏"逻辑→"滞胀情景"（当前处于通胀预期阶段）
2. **霍尔木兹海峡局势**：船只数量锐减，油价维持高位，短期难以解封
3. **美联储态度**：FOMC偏谨慎，点阵图朝"减少降息次数"方向变化
4. **就业市场疲弱**：非农负增长+AI替代担忧升温

## 未来可能出现的现象

- 若中东事态升级+3月数据证实"经济走弱+通胀抬升"，市场将向滞胀情形切换
- 4月可能在通胀预期和滞胀预期之间反复摇摆
- 油价走势取决于霍尔木兹海峡封锁何时解除（乐观1个月/中性1季度/悲观数月）
- 若滞胀显性化，美债利率大趋势仍向上，但期间可能因衰退预期摇摆而阶段性回落
- OPEC可能顺势控价，70年代油价中枢长期抬升或重演

## 投资含义

| 资产 | 4月建议 | 核心逻辑 |
|------|---------|----------|
| 美债 | 谨慎，利率难回落 | 通胀预期强化+降息延后，期限利差可能收窄 |
| 美股 | 短期避险为主 | 标普500/纳指跌破支撑位，长期AI产业趋势值得博弈 |
| 商品 | 易涨难跌 | 供给端逻辑+供应链补库+AI能源需求+产能调控 |
| 黄金 | 长期配置机会 | 短期流动性冲击提供买点，滞胀阶段实际利率下行将驱动金价 |
| A股 | 区间震荡 | 避险情绪主导，结构关注三条主线 |
| 港股 | 恒生科技相对弱势 | 外资持仓占比高，受流动性负反馈 |

## A股三条结构性主线

1. **算电协同**：政府工作报告首提，绿电/储能短期支撑更强
2. **反内卷**：化工、光伏、新能源汽车；政策加速+能源成本抬升强化涨价逻辑
3. **红利防御**：金融、公用事业等防御型板块

## 作者主观观点压缩版

- 中东局势是当前市场的定价主线，事态还有进一步发酵可能
- 美联储政策传导效率高于70年代，市场直接跳跃至通胀逻辑
- 美股虽受滞胀负反馈，但AI科技龙头驱动+ROE上行阶段可呈现韧性（参考1978-1980）
- 黄金短期下跌反而是长期配置良机，等待4月观察窗口
- A股等待增量资金信号，近期成交额一度跌破2万亿

## 风险提示

1. 海外通胀反弹超预期
2. 全球经济景气度不及预期
3. 地缘政治局势超预期
