# 西南证券2026年3月PMI数据点评：制造业PMI重返扩张区间，需求端改善显著

## 一句话结论

制造业PMI时隔两个月重返荣枯线以上（50.4%），需求端改善强于生产端，新动能产业与小型企业反弹最为突出，但外需受中东地缘冲突拖累与成本压力需持续关注。

## 最重要的边际变化

1. **需求改善强于生产**：新订单指数回升3pp至51.6%，高于生产指数0.2pp，产需缺口收窄
2. **小型企业景气大幅反弹**：PMI回升4.5pp至49.3%，对市场变化反应更敏捷
3. **价格指数跳升**：原材料购进价格指数大幅回升9.1pp至63.9%，受中东地缘冲突推升国际油价影响
4. **新出口订单大幅回升**：4.1pp至49.1%，但仍处收缩区间，外需仍有压力
5. **企业采购活跃度提升**：采购量指数回升2.7pp至50.9%，重回扩张区间

## 未来可能出现的现象

1. **4月PMI季节性回调**：节后集中开工高峰过去，生产经营活动恢复正常节奏
2. **地缘冲突持续推高成本**：中东局势扰动全球能源供应链，原材料价格保持高位震荡
3. **高技术制造业保持韧性**：新动能领域预计持续为整体景气度提供支撑
4. **基建投资稳步回升**：8000亿元超长期国债与新型政策性金融工具逐步形成实物工作量
5. **建筑业景气延续改善**：气温回升带动工地施工条件改善，进入传统施工旺季
6. **欧洲央行或加息**：欧央行行长表态将果断应对能源价格冲击，可能抑制欧元区制造业

## 投资含义

1. **制造业复苏信号确认**：PMI重返扩张区间，政策效果开始显现，内需支撑增强
2. **警惕成本压力传导**：原材料价格大幅上涨，若无法有效传导至终端需求，可能挤压中下游企业利润
3. **关注小企业修复持续性**："春雨润苗"专项行动精准发力，政策支持效果待观察
4. **外需不确定性**：中东地缘冲突、加征关税持续影响出口，对制造业景气度构成压制
5. **新动能仍是主线**：高技术制造业与装备制造业持续领先，关注AI相关领域出口动能
6. **地产需求边际改善**：30城商品房成交同比降幅较上月显著收窄，但投资端仍承压

## 作者主观观点压缩版

西南证券认为，3月PMI数据释放积极信号，制造业景气重返扩张区间，供需两端协同回暖。**积极因素**：扩内需政策持续发力（2500亿超长期特别国债、1000亿促消费资金、8000亿"两重"建设）；新动能产业（高技术、装备制造）连续14-20个月新订单扩张；小型企业反弹弹性大。**风险因素**：中东地缘冲突推升国际油价，抬高制造业成本、拖累外需；欧洲央行若加息将抑制外部需求；企业利润前瞻指数下降反映成本压力担忧。**展望**：4月PMI面临季节性回调压力，但政策持续发力背景下，制造业景气向好态势有望延续，PPI降幅预计继续收窄。
