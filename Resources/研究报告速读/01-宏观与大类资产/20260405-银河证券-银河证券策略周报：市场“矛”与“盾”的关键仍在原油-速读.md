# 市场"矛"与"盾"的关键仍在原油
# 银河证券策略周报 2026-04-05

## 一句话结论

## 最重要的边际变化

1. **资金大幅撤离**：全球基金对A股净流出-28.08亿美元（前值+3.55亿美元），由净流入转为净流出。海外基金净流出-10.69亿美元（被动型-9.68亿，主动型-1.05亿）。

2. **科技板块被抛售**：全球基金对科技行业净流出-14.02亿美元（前值+11.93亿），由净流入转为净流出。海外基金科技净流出-4.71亿美元。

3. **美伊冲突升级**：特朗普划定4月6日为最后期限，伊朗反击烈度与范围扩大，地缘风险持续压制风险偏好。

4. **交投显著降温**：日均成交额较上周下降超2000亿元，换手率下降0.27个百分点。

5. **开户数据仍旺**：3月A股新开户460万户，环比+82.4%，同比+50.1%；2026年累计新开户1204万户，同比+61.2%。

## 未来可能出现的现象

1. **4月业绩披露压力**：年报、一季报进入集中披露期，业绩验证将成为市场主线。

2. **原油价格仍是关键**：若冲突缓和→油价回落→宽松预期回升→利好成长股；若冲突持续→通胀预期升温→压制科技。

3. **限售股解禁压力下周上升**：下周解禁市值约1089亿元，较本周378亿元大幅增加。

4. **防御风格可能延续**：在高波动环境下，业绩确定性高、景气度持续的板块更受资金青睐。

## 投资含义

**三大配置方向**：

| 方向 | 细分领域 | 逻辑 |
|------|---------|------|
| 能源及替代 | 煤炭、煤化工、新能源、航运港口、油气、有色金属 | 美伊冲突驱动能源及替代性需求 |
| 防御资产 | 银行、公用事业、交通运输 | 阶段性占优，对冲风险 |
| 科技成长 | 电力设备、储能、存储、半导体、算力、通信设备 | 产业趋势确定性，中长期主线 |
| 消费修复 | 农林牧渔、食品饮料、家用电器 | 估值处于历史偏低，具备修复空间 |

**中长期双主线**：科技板块产业驱动 + 周期板块涨价线索

## 作者主观观点压缩版

- **外部风险是最大扰动**：美伊冲突不确定性仍高，全球权益市场高波动延续，A股呈震荡轮动特征。
- **核心逻辑未变**：政策支撑、居民入市、中国资产重估的慢牛根基未被动摇，外资短期撤离不改中长期趋势。
- **油价是关键矛盾**：原油通过通胀→流动性→市场结构的传导链，成为近期市场分化的核心驱动。
- **业绩验证将主导4月**：市场将从主题炒作逐步转向基本面验证，业绩确定性是筛选方向。
- **提示风险**：外部不确定性、政策不及预期、市场情绪不稳定。
