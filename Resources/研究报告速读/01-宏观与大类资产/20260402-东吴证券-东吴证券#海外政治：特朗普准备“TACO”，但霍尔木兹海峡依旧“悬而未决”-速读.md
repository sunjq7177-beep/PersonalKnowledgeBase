# 海外政治：特朗普准备"TACO"，但霍尔木兹海峡依旧"悬而未决"

## 一句话结论

特朗普或通过最后一波对伊朗高烈度军事打击宣布"彻底胜利"，以正当化撤军的TACO行为；但无论"升级路径"还是"TACO路径"均无法解决霍尔木兹海峡通航问题，市场短期将延续"紧货币"交易路径。

## 核心数据卡

**市场反应**：演讲结束后原油价格大涨，美债利率上行，黄金价格回落。

## 最重要的边际变化

1. **特朗普4月2日白宫演讲**：3月3日以来第二次就对伊军事行动发表正式演讲，核心目标"接近完成"，将再度"严厉打击"伊朗
2. **霍尔木兹海峡"甩锅"**：称美国几乎不从该海峡进口石油，要求其他国家"夺取并保护"海峡，宣称冲突结束后海峡将"自动开放"
3. **伊朗革命卫队强硬回应**：表示海峡仍处伊朗控制之下，"绝不会向敌人开放"
4. **TACO准备迹象**：美军大规模调兵（双航母+两支海军陆战队+空降师），为地面攻击后迅速撤军创造条件

## 未来可能出现的现象

1. **高烈度地面攻击**：美军在双航母空中支援下，海军陆战队+空降部队对伊朗发动一轮地面打击后迅速撤退
2. **宣布"史诗狂怒"胜利**：特朗普宣布完成所有战略目标打击，获得"彻底胜利"
3. **撤军但不解决海峡问题**：即便美军撤出，霍尔木兹海峡仍可能仅允许伊朗友好国家船只通过
4. **矛盾转移**：特朗普将海峡封锁责任归咎于伊朗，试图掩盖自身挑起冲突的政治责任
5. **伊朗若积极表态**：若伊朗对美伊直接会谈做出积极回应，双方开始和平协议磋商，市场情绪可能显著改善

## 投资含义

**短期路径**：延续"紧货币"交易
- 风险资产承压
- 黄金价格回落
- 美债利率上行
- 原油价格大涨

**反转条件**：伊朗方面对美伊直接会谈做出积极表态，双方开始和平协议实质性磋商，则"紧货币"交易路径可能反转。

## 作者主观观点压缩版

东吴证券认为，特朗普对伊军事行动本质上是希望复刻委内瑞拉"成功案例"，通过斩首行动迫使伊朗让步，但伊朗在哈梅内伊身亡后反而强硬反击，导致局势超出预期。当前油价飙升令特朗普面临国内经济和选民双重压力，"直接撤军"等于承认战略误判，因此选择"先升级再降级"的TACO路径。关键问题在于：撤军后霍尔木兹海峡通航问题仍无法解决，伊朗大概率保持控制以换取战后利益。市场短期将延续当前交易路径，直至美伊出现实质性谈判信号。
