# 论中国式产业周期

## 一句话结论

中国产业周期与"五年规划"深度绑定，每轮周期（约10-20年）遵循"萌芽→成长→整合→成熟"四阶段演化，当前以新质生产力为核心的科技创新周期正处于从萌芽向成长纵深发展的关键阶段。

## 核心数据卡

| 维度 | 数据 |
|------|------|
| **重化工业周期** | "十五"期间基建投资增速均值20.2%，同期GDP增速9.5%；钢铁煤炭市值占比一度达6%-9% |
| **移动互联网周期** | 2002-2011年电子信息制造业增加值增长6倍，销售收入年均增速21.4%，出口年均增速24.5% |
| **消费升级周期** | "十二五"至"十三五"居民人均可支配收入实际年均增速7.2%，社消总额从17.98万亿增至39.05万亿（+117.2%） |
| **新能源汽车** | "十三五"设定累计产销量500万辆目标，现连续多年全球第一 |
| **当前资本开支** | 半导体行业研发支出持续攀升，一级市场AI/半导体投融资活跃 |
| **中国神华案例** | IPO募资665.82亿+发债700亿=1218.6亿，累计分红4804.7亿，股息率7.07%（vs万得全A的1.74%） |

## 最重要的边际变化

1. **主线切换信号**：经济增长内核从"投资驱动→消费驱动→创新驱动"的深层结构性变革
2. **A股龙头变迁**：从周期性行业（材料、能源、工业）→大金融→"十四五"后TMT和周期性行业均衡分布
3. **估值逻辑转变**：硬科技企业估值从看重当期盈利转向包含成长预期定价
4. **当前周期特征**：新质生产力正在接替传统消费与投资，成为经济增长新支柱

## 未来可能出现的现象

1. **一级市场**：资金持续密集投向人工智能、半导体、商业航天等硬科技领域
2. **二级市场**：相关行业研发投入与资本开支强度进入快速攀升通道，预示产能扩张与技术商业化加速
3. **产业分化**：真正实现技术突破与商业落地的企业逐步突围，市场表现开始分化
4. **出清信号**：关注毛利率与产能利用率等指标对景气拐点的敏感性
5. **投资转移**：科技创新产业接替传统消费与投资，成为经济增长新支柱

## 投资含义

1. **战略定位**：产业周期回答"身处何处"的战略问题，决定长期增长中枢、竞争格局及核心定价逻辑
2. **周期节奏**：萌芽后期至成长初期市值扩张斜率最陡峭；整合期市值占比普遍持续下滑；成熟期盈利稳定板块吸引力回升
3. **五年规划指引**：政策导向构成产业景气抬升先行指标，"规划定调—资本响应—产业加速"构成中国特色范式
4. **长短周期嵌套**：忽视产业长周期坐标、单纯依赖短周期指标线性外推，可能导致投资节奏误判
5. **核心考量**：技术路线选择与商业化落地能力成为当前硬科技估值核心要素
6. **风险提示**：宏观经济波动、行业竞争与政策变动、科技创新不及预期

## 作者主观观点压缩版

1. **周期认知**：中国式产业周期发展节奏较快，从萌芽到成熟时间并不长，需及时发现、抓住周期
2. **规划价值**：每一波产业周期的崛起几乎都能从"五年规划"中找到参考答案，政策研究具有重要前瞻性意义
3. **主线判断**：以技术和安全为底色的新质生产力产业周期正在接替传统动能，前瞻布局未来十年投资机遇的战略意义重大
4. **企业逻辑**：一轮产业周期不仅能引领市场行情，更能诞生一批优秀上市公司；当前AI新周期中，上一轮周期崛起企业通过业务重新布局，有望迎来发展"第二春"
