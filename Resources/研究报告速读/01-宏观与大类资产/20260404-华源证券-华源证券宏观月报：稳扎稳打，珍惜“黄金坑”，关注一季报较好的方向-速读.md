# 华源证券宏观月报：稳扎稳打，珍惜"黄金坑"，关注一季报较好的方向

## 一句话结论

4月市场以震荡为主，建议珍惜"黄金坑"，积极布局能源主线及一季报超预期方向，关注美伊停战带来的地缘红利。

## 最重要的边际变化

1. **地缘转向**：美伊冲突4月有望停战，霍尔木兹海峡通航带来地缘红利，欧洲转向+一带一路配合度提升
2. **出口韧性**：出口推动PMI、制造业，财政支出支撑基建，关注二季度持续性
3. **政策落地**：两会、"十五五"规划相继落地，科技、自主可控投资延续
4. **能源价格高位**：原油118美元/桶，沥青月环比+37%，焦煤+12.5%
5. **防御资产分化**：黄金环比-11%但仍同比+49%，消费、高股息成避风港

## 未来可能出现的现象

1. **5月中美和谈前**：局势和风险偏好有望重新回归
2. **能源价格波动**：短期油价仍高，后续伴随较大风险
3. **一季报验证**：AI算力、国产芯片、半导体设备景气度分化将显现
4. **海外关注点**：美国私募信贷风险、AI泡沫可持续性、鲍威尔5月任期结束
5. **存储涨价**：内存价格较2025年末已上涨80%-90%

## 投资含义

### 进攻方向
- **能源主线**：新旧能源持续看好，油价100美元下化工产业链仍有盈利空间
- **AI与半导体**：关注一季报延续高增长标的，景气度高位但需验证
- **出海链**：一带一路发展中国家配合度提升

### 防御方向
- **黄金**：各国储备需求，长期逻辑不变
- **消费**：社零增速稳健，必选消费韧性
- **高股息**：市场震荡期确定性溢价

### 风险监控
- 国内：物价低位运行、地产销售下行、出口贸易政策
- 海外：地缘冲突加剧、美联储降息不及预期、海外衰退

## 作者主观观点压缩版

报告认为**现阶段战争局势对中国经济影响不大，部分行业甚至偏利好**。建议投资者保持耐心珍惜"黄金坑"，核心逻辑：
1. 美伊停战→地缘红利→出口+一带一路受益
2. 政策托底→科技投资延续→制造业韧性
3. 能源价格高位→短期扰动但盈利空间仍存
4. 市场震荡→防御为主+一季报验证后进攻
