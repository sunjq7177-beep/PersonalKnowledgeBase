# 外部扰动下的"四月决断"
## 一句话结论
地缘冲突引发市场调整，但A股相对海外资产更具稳定性；进入4月应抓住"四月决断"窗口期，从预期驱动转向现实定价，配置上采取"泛能源+科技缩圈"对冲策略。

## 核心数据卡

| 指标 | 数据 | 变化/意义 |
|------|------|----------|
| 融资余额 | 2.58万亿 | 较3月初下滑760.77亿，杠杆风险水平不高 |
| ETF总份额 | 2.10万亿份 | 较3月初下滑119.39亿份，市值缩水非赎回 |
| 成交量 | 1.67万亿 | 未低于2025年12月低点 |
| 3月新增开户数 | 460万户 | 仅次于2024年10月和2026年1月 |
| 2月出口当月同比 | +39.6% | 1月为+10%，出口强劲 |
| 规上产成品存货同比 | +6.6% | 2023年以来最高增速 |
| 2月PPI当月同比 | -0.90% | 1月-1.40%，持续改善 |
| 油价中枢 | 100-110美元/桶 | 较冲突升级前明显上移 |

## 最重要的边际变化

1. **油价中枢上移**：中东局势推动油价升至100-110美元/桶区间，中期传导至通胀链条，海外尤其美国面临"类滞胀"风险
2. **出口超预期**：1-2月经济数据亮眼，2月出口同比+39.6%，带动库存周期修复
3. **A股相对稳健**：地缘冲击以来A股相对美股、日经、韩股更具稳定性
4. **弱美元叙事转向**：高油价持续冲击降息预期，流动性收紧预期升温
5. **居民热情未减**：新增开户数460万户（历史高位区间），投资者持观望而非恐慌

## 未来可能出现的现象

1. **海外滞胀风险升温**：若油价持续高位，中期传导至通胀链条，美国可能面临滞胀压力
2. **"弱美元"叙事转向**：流动性收紧预期升温，高估值、高杠杆资产或遭遇冲击
3. **市场结构性切换**：从科技有色等"弱美元"受益板块，向泛能源、绩优标的切换
4. **地缘"边打边谈"**：美伊局势反复拉锯，市场情绪交替轮动
5. **基本面定价回归**：4月业绩集中披露期，缺乏业绩支撑的高位板块面临调整
6. **居民加速入市**：若地缘风险缓和，观望资金可能快速入场

## 投资含义

1. **4月是关键验证窗口**：经济数据成色、业绩兑现能力、政策边际变化三重验证
2. **规避高位兑现周期长板块**：风险收益呈非对称特征，高估值资产承压
3. **泛能源是滞胀最优选择**：历史上海外滞胀环境中表现最好，关注电力、煤化工、电网设备
4. **新能源具备双重支撑**：能源转型结构性优势+产业景气，发电侧经济性提升+储能技术领先
5. **科技需"缩圈"精选**：有护城河、有定价权、有业绩的细分领域才能穿越周期
6. **地产链仍需观察**：一线城市二手成交回暖但尚不构成趋势性

## 作者主观观点压缩版

东吴证券认为，当前地缘冲突正边际升级但仍可控，市场处于"宏微观博弈"阶段。4月是全年最重要的决断窗口，需从预期驱动转向基本面定价。建议采取"泛能源+科技缩圈"对冲策略——泛能源应对滞胀风险、享受能源替代逻辑；科技精选有业绩支撑的细分领域。具体看：
- 泛能源：电力、煤化工、电网设备、光伏、储能、锂电池、风电、核电
- 科技缩圈：供需矛盾突出、具备业绩支撑的细分科技领域

核心风险：国内复苏不及预期、联储降息不及预期、地缘政治风险。
