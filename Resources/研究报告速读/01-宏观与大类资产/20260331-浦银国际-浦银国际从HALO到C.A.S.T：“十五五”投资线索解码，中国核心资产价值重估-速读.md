# 从 HALO到 C.A.S.T：“十五五”投资线索解码，中国核心资产价值重估

**来源**：浦银国际研究 | 2026年3月31日

## 一句话结论

中国HALO资产核心价值在于“体系完整、估值洼地、政策托底”下的优势资产价值重估，C.A.S.T.框架（核心资产+AI基建+稳定现金流+技术壁垒）结合杠铃策略，防御端聚焦央国企重资产，成长端捕捉AI算力基建超级周期。

## 最重要的边际变化

**“十五五”与“十四五”规划对比的核心差异**：

1. **能源体系排序前移**：加力建设新型能源基础设施从第三位升至第二位，非化石能源目标首次纳入经济社会发展目标
2. **交通运输重回首位**：完善现代化综合交通运输体系排基建规划首要位置，高铁、高速公路、港口、机场仍是重点
3. **都市圈轨道重要性下降**：城市群和都市圈轨道交通在基建规划中未提及，仅在城市更新部分略谈
4. **地下管网成为新型城镇化首要任务**：预计建设改造超70万公里，投资超5万亿元
5. **新型基建措辞减弱**：从“加快建设”变为“适度超前建设”，排序从首位降至末位
6. **特高压建设加码**：十五五规划投运15项特高压直流工程，跨省输送能力提升35%

## 未来可能出现的现象

1. **算电协同商业落地**：AI数据中心可调负荷占比10%-25%，虚拟电厂聚合调峰调频收益，算电协同配套市场规模2030年突破万亿级别
2. **变压器出口持续放量**：海外AI数据中心建设提速，北美变压器短缺交付周期2-4年，中国供应链显著受益，部分龙头企业海外订单已排至2027年
3. **全国一体化算力网加速**：5G-A基站建设（规划50万座）、6G技术储备、东数西算工程深化
4. **雅鲁藏布江下游水电开工**：总投资1.2万亿元，澜沧江上游、藏东南等流域水风光一体化基地首次提及
5. **电价机制改革深化**：容量电价机制为火电构筑盈利底线，统一电力市场体系建设推进
6. **AI应用重构电力需求**：一次大模型查询耗电量为传统搜索10倍，全球数据中心2030年用电量较2024年翻倍

## 投资含义

**C.A.S.T.框架下的投资主线**：

1. **防御端（杠铃左端）**：
   - **有色金属**：铜、铝、锂、黄金、稀土，受益AI基建新增需求+供给约束
   - **能源**：石油石化、煤炭，高股息+能源安全底座+PPI回升弹性
   - **电力**：水电/核电，长周期稳定现金流+AI算力需求重估+出海逻辑
   - **交通运输**：铁路、港口、管道，网络效应+自然垄断+稳定ROE

2. **成长端（杠铃右端）**：
   - **光模块**：800G/1.6T规模化量产，全球供应链主导，业绩持续兑现
   - **高端PCB**：AI服务器高多层/HDI需求，国内厂商规模优势
   - **液冷散热**：AI芯片功耗攀升，国产微通道/冷板规模化交付
   - **算力芯片**：华为昇腾/阿里平头哥推理侧加速替代，国产化率快速提升
   - **变压器/特高压**：出海订单排至2027年，十五五电网投资4万亿量级

3. **电力行业重点细分**：
   - 变压器（干式/特种变压器）：AI数据中心需求激增+原材料自给优势
   - 特高压设备：十五五15项直流工程+跨省输送能力提升35%
   - 配电端设备：单机柜功率提升驱动高端化升级
   - 储能电力电子：SST/PSU需求+虚拟电厂运营

4. **估值逻辑**：中国公用事业市净率1.09倍vs美国2.16倍，估值修复空间显著；低利率环境下高股息吸引力凸显；政策端分红与市值管理引导强化配置价值

## 作者主观观点压缩版

**核心判断**：

- 中国HALO资产与美国HALO资产本质不同：美国聚焦存量稀缺性重估，中国核心在于“体系完整+估值洼地+政策托底”下的优势资产价值重估
- 杠铃策略是中国市场最优解：防御端央国企高股息低估值穿越周期，成长端AI算力基建业绩兑现捕捉弹性
- 电力和通信是C.A.S.T.框架最具代表性的行业：既是国家战略底座（Core Assets），又是AI算力核心载体（AI Infrastructure），同时具备重资产壁垒稳定现金流（Stable Cash Flow）和技术护城河（Tech Moats）
- 十五五期间传统基建和能源基建重要性提升，但AI算力建设不会减速——美国限制加速国产替代，国内算力基建加速部署是应对外部挑战的必然选择
- 估值重塑需盈利驱动：PPI回升对上游资源品盈利形成支撑，但关键在于终端需求实质扩张，否则易陷入“量缩价涨”格局
- AI对电力的影响不仅是需求侧：算电协同商业模式（虚拟电厂调峰、数据中心参与辅助服务）正在重塑电力行业价值链
