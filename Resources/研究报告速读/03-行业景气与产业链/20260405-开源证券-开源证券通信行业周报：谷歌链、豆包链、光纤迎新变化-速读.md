# 谷歌链、豆包链、光纤迎新变化——通信行业周报

## 一句话结论

谷歌 AI 大会密集举行叠加豆包 Token 消耗量爆发，全球光纤光缆量价齐升，AI 基础设施产业链三大主线（光、液冷、国产算力）持续受益。

## 最重要的边际变化

1. **Google Cloud Next '26（4.22-24）**：新一代 TPU 架构发布在即，OCS 光路交换机实际应用落地，有望打开 OCS 增量空间
2. **Google I/O 2026（5.19-20）**：Android 17 预览、AI 眼镜亮相，AI 或从独立产品渗透为基础设施
3. **豆包大模型 2.0 系列**：多模态理解、复杂任务执行跻身全球第一梯队，成为国内唯一双认证厂商（智能体产品可信 + 安全防护）
4. **光纤价格突破历史周期高点**：G652.D 裸光纤 83.40 元/芯公里超前期峰值，AI 数据中心驱动非周期性涨价，供需缺口或持续至 2027 年

## 未来可能出现的现象

1. **OCS 光路交换机规模化部署**：谷歌 Cloud Next 或成新技术路线起点
2. **AI 眼镜生态加速**：Gemini 模型落地硬件，人机交互模式变革
3. **Token 调用量持续爆发**：国家数据局预测国内日均 Token 两年增长超千倍，豆包 Seedance 2.0 海外上线后增长空间巨大
4. **光纤价格中枢维持高位**：供需缺口扩大至 15%，"棒-纤-缆"一体化扩产周期长，价格偏紧或成常态
5. **字节跳动海外算力需求释放**：当前 Token 消耗主要来自中国市场，视频生成模型 API 尚未出海

## 作者主观观点压缩版

开源证券认为：AI "虹吸效应"显著，2026 年全球 AI 或继续共振。谷歌、Meta 等海外巨头持续上调 AI 资本开支，Gemini Tokens 消耗量大幅提升；字节、阿里、腾讯等国内 AI 巨头进入算力大规模投入期。光纤涨价非简单周期性波动，而是 AI 数据中心等新兴需求驱动，预计供需缺口持续扩大。Token 正成为新的基础资源，国内 Token 调用量两年增长超千倍意味着 AI 产业落地全面加速。相对独立产品，AI 有望从各方面渗透存量产品并成为基础设施，催生更大算力需求。维持行业"看好"评级。
