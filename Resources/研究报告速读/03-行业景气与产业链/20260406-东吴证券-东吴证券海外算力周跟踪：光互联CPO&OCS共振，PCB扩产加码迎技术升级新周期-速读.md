# 海外算力周跟踪：光互联CPO&OCS共振，PCB扩产加码迎技术升级新周期

## 一句话结论

AI算力基础设施进入技术路线收敛期，光互联（CPO/OCS）从概念验证迈向规模商用，PCB龙头同步扩产高端产能，2027年Rubin Ultra将成为最强需求催化。

## 最重要的边际变化

**1. CPO商业化元年开启**
- 台积电4月1日宣布硅光整合平台COUPE年内量产
- 英伟达GTC 2026明确将CPO作为AI算力集群功耗/带宽瓶颈破解关键路线
- OFC 2026：Lightmatter发布可拆光纤阵列vClick™，Coherent推出6.4T socketed CPO方案
- 国内：长光华芯+亨通光电联合启动全国首条8英寸90nm硅光芯片量产线

**2. OCS市场预期大幅上修**
- Coherent将2030年全球市场从20亿→40亿美元翻倍上调
- 工信部4月2日普惠算力专项行动明确推动全光交换技术部署
- 新易盛NX200/NX300系列分别支持140/320端口，MEMS微镜阵列实现亚微秒级光路切换

**3. PCB扩产全面聚焦AI刚需品类**
- 三大龙头2026年合计资本开支超544亿元
- 低端产能几无扩张，18层以上高多层板、高阶HDI成扩产重心
- 行业结构加速向高附加值升级

## 未来可能出现的现象

1. **2026年**：CPO器件批量出货，OCS在AI算力集群中大规模部署
2. **2026H2**：Rubin Ultra进入设计定稿阶段，PCB厂商开始备货高多层板
3. **2027年**：Rubin Ultra量产落地，带动MSAP/SLP精密工艺需求爆发
4. **2027-2028年**：OCS市场从Spine层向Scale-up/Scale-out场景全面渗透
5. **2029年**：全球OCS市场突破25亿美元，年复合增速维持58%高位

## 投资含义

1. **光互联产业链**：CPO/OCS从主题投资进入业绩兑现期，核心器件厂商订单可见度提升
2. **PCB行业**：扩产龙头（胜宏、鹏鼎、沪电）绑定AI芯片升级周期，2027年需求催化明确
3. **设备端**：硅光耦合设备（罗博特科）直接受益CPO扩产，6亿元订单为起点
4. **材料端**：MSAP/SLP精密工艺能力将成为高阶PCB核心壁垒

## 作者主观观点压缩版

分析师维持**增持**评级，核心逻辑：

- CPO/OCS共振代表AI算力互联的底层技术革命，国内产业链（长光华芯、亨通、罗博特科）同步跟进
- PCB扩产是机构投资者用真金白银投票验证行业高景气，供需格局持续优化
- Rubin Ultra的CoWoP封装路线省去传统基板，单板价值量2-3倍提升，高多层+精密工艺双重受益
- 风险在于供应链波动和下游需求不及预期，但行业β明确

**一句话**：2026年是CPO商业化元年，2027年是PCB新一轮增长周期起点，当前是布局窗口期。
