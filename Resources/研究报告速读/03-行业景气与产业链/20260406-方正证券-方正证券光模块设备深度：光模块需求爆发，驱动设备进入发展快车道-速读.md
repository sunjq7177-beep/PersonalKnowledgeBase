# 光模块设备深度：需求爆发驱动设备进入发展快车道

## 一句话结论
AI算力需求爆发推动高速光模块快速迭代，800G设备已成主导，1.6T即将放量，光模块封测设备市场2020-2024 CAGR达71.8%，2024年迈入放量元年。

## 核心数据卡

| 指标 | 数值 | 备注 |
|------|------|------|
| 2024年全球光模块市场 | 178亿美元（1246亿元） | - |
| 2029年预计规模 | 415亿美元（2905亿元） | 2024-2029 CAGR约18% |
| 2024年光模块封测设备市场 | 51.8亿元 | 2020-2024 CAGR 71.8% |

| 固晶机2024年规模 | 9.8亿元 | 占比18.9%，2029年预计24亿元 |
| 光耦合机2024年规模 | 12.1亿元 | 占比23.3%，2029年预计22.7亿元 |
| 老化测试设备2024年规模 | 16.3亿元 | 占比31.4%，最高价值占比设备 |
| 1.6T光模块CAGR | 180% | 2024-2029 |
| 硅光智能制造设备CAGR | 34.1% | 2024-2029 |

## 最重要的边际变化

1. **800G设备爆发**：从2022年0.1亿元激增至2024年30.2亿元，两年增长300倍，成为增长最快细分领域
2. **生产模式转型**：过去多批次小批量→当前大规模自动化组装元年，设备需求性质根本改变
3. **价值量集中**：共晶/固晶机+老化测试合计占比接近80%，是设备投资核心方向
4. **代际升级加速**：400G→800G→1.6T升级周期从4年缩短至2年，技术迭代倒逼设备持续升级

## 未来可能出现的现象

1. **2026-2027年**：1.6T进入主流应用，CPO/NPO开始规模上量
2. **2027-2028年**：CPO及OCS技术进入大规模应用，带动硅光制造设备加速增长
3. **2029年**：3.2T光模块开始预研，CPO在3.2T市场渗透率达50.6%
4. **硅光占比提升**：硅光制造设备占光子制造设备份额将从24年24%→29年63%→35年80%

## 作者主观观点压缩版

光模块产业已从电信驱动全面转向AI算力驱动，技术迭代周期从4年压缩至2年。800G是当前主流放量品种，1.6T是下一阶段核心战场，3.2T开始预研。设备厂商的机遇在于：①高速率迭代带来持续设备更新需求②国产化替代打破海外垄断③CPO/硅光等新技术路线带来弯道超车机会。核心关注具备高精度、纳米级制造能力的设备商，以及能覆盖多工艺环节的平台型公司。
