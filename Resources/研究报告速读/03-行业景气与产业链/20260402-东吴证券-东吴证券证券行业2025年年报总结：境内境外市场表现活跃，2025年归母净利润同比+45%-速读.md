# 证券行业2025年年报总结：境内境外市场表现活跃，2025年归母净利润同比+45%

## 一句话结论

经纪与自营双轮驱动，上市券商净利润同比+45%至1856亿元，但Q4环比-30%反映高基数压力；当前估值处于历史底部区域，大型券商凭借资本实力和项目优势集中度持续提升。

## 核心数据卡

| 指标 | 2025年 | 同比变化 |
|------|--------|----------|
| 日均股基交易额 | 2.05万亿元 | +70% |
| 融资余额 | 2.5万亿元 | +36% |
| IPO规模（A股） | 1318亿元 | +96% |
| 再融资规模 | 9509亿元 | +326% |
| 港股IPO规模 | 2869亿港元 | +225% |
| 上市券商营业收入 | 4564亿元 | +32% |
| 上市券商归母净利润 | 1856亿元 | +45% |
| 平均ROE | 7.47% | +1.88pct |
| 上市券商自营规模 | 4.5万亿元 | +13% |
| 分红总额 | 577亿元 | +33% |

**分部收入结构**：经纪27%、投行7%、资管9%、利息9%、自营42%、其他6%

## 最重要的边际变化

1. **IPO触底回升，集中度飙升**：A股IPO 116家（同比+96%），港股IPO 117家；境内IPO CR5从55%跃升至73%
2. **自营持仓结构调整**：券商增配OCI权益工具，其他权益工具规模6195亿元（同比+57%），较2021年末增10倍
3. **费率压力显现**：经纪收入增速(+45%)远低于交易额增速(+70%)，主因量化/ETF低费率交易占比提升+基金费改
4. **Q4高基数效应**：Q4净利润392亿元，同比+8%但环比-30%，主要来自自营环比-46%
5. **分红比例持续提升**：平均分红比例31%，同比+2pct；H股平均股息率4.0%

## 未来可能出现的现象

- **2026年Q1高基数检验**：年初日均股基交易额接近3.1万亿元（较2025年高基数仍+53%），3月末两融余额2.6万亿元（较年初+3%）
- **投行集中度或进一步提升**：IPO向"少而精"转型，科创板/北交所更依赖头部定价、风控与承销能力
- **券商再融资全面放开**：大股东支持力度加大，资产驱动扩表逻辑持续
- **基金费率改革深化**：公募基金降费趋势延续，倒逼基金公司产品结构优化

## 投资含义

1. **估值修复空间显著**：中信证券II指数PB 1.23x，处于近十年13%分位（历史底部）
2. **大型券商应享估值溢价**：政策鼓励并购做优做强，马太效应加剧，CR5/CR10净利润占比60%
3. **盈利预测**：预计2026年净利润同比+15%，其中经纪+23%、投行+10%、资本中介+14%、资管+10%、自营持平
4. **重点标的**：中信证券、华泰证券、国泰海通、广发证券、兴业证券、财通证券、东方财富、指南针、湘财股份

## 作者主观观点压缩版

- 2024年9月"924行情"开启的交投活跃度具备持续性，非短期脉冲
- 费类业务(经纪+投行+资管)是ROE提升的核心驱动力，自营提供弹性
- 券商从通道中介向资本中介转型的长期逻辑确认，重资产能力差异决定估值分化
- 散户入场(沪市月均开户250万户)+机构化(ETF扩容)并存，经纪业务α在交易量而非佣金率
- 债券市场虽弱，但头部券商债券类自营以配置盘为主，亏损有限
