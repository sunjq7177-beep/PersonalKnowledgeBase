# 东吴证券电力设备行业周报：油价高锂电需求向好，北美缺电持续高景气

## 一句话结论

锂电产业链景气上行，碳酸锂价格回落带动电池成本传导顺畅；北美AI驱动大储需求爆发，电力设备出海持续受益；光伏产业链价格持续承压，人形机器人量产元年开启。

## 最重要的边际变化

1. **锂电产业链价格企稳回升**：碳酸锂价格已至底部区间（14万/吨），津巴锂矿出口恢复仍需时间；电池级散单价格已调至0.38元/wh，材料中小客户涨价已落地，大客户预计4月逐步落地

2. **北美大储需求爆发**：2026年预计美国大储装机80GWh（+51%），数据中心贡献37GWh；关税及"大而美"法案落地，25-26年发货景气度高

3. **排产持续改善**：3月环比+20%后，4月环增5%，5月预计环比增10%左右，景气持续上行

4. **变压器等电力设备招标高增**：国家电网第6批招标中，变压器+110%、组合电器+52%、电抗器+61%，一次设备整体稳健增长

5. **光伏产业链价格持续下行**：硅料-13.95%领跌，组件价格面临拐点压力，预计4月下旬可能松动

## 未来可能出现的现象

1. **4月电动车销量同比有望转正**：3-4月新车发布，订单边际好转，预计全年本土销量增长5%，出口增长50%

2. **碳酸锂价格高位持续**：津巴锂矿出口恢复仍需时间，碳酸锂价格高位震荡，材料大客户涨价预计4月逐步落地

3. **光伏组件价格4月下旬可能进入拐点**：需求疲弱+成本下行通道，组件价格将面临下行压力

4. **美国大储2026年装机80GWh**：数据中心贡献37GWh，中国供应链有望借势美国光储高增

5. **Tesla Gen3将于2026年4月推出**：人形机器人量产过渡期开启，2027年量纲是关键验证节点

6. **容量补偿电价细则陆续出台**：国家层面政策落地，电算协同进一步提振储能需求

## 作者主观观点压缩版

1. **锂电板块最底部区域已过**：排产环增20%持续，碳酸锂价格高位+电池价格传导顺畅，4月大客户涨价落地+盈利拐点确认，首推电池和隔膜

2. **储能是最景气的赛道**：26年全球装机60%+增长，27-29年CAGR 30-50%，美国大储+欧洲户储+中东大项目多点开花

3. **北美缺电是长期主题**：AI算力升级+制造业回流+新能源渗透率提升，变压器、SST、燃气轮机持续受益

4. **人形机器人处于0-1阶段**：当前相当于2014年电动车，Tesla引领，100万台仅是起点，远期空间上亿台

5. **光伏短期承压但新技术有看点**：组件价格面临拐点，但太空光伏（Musk计划100GW太空+100GW地面）带来增量想象空间

6. **风电景气周期开启**：欧洲能源安全推动海风加速，26年国内海风有望重拾高增长
