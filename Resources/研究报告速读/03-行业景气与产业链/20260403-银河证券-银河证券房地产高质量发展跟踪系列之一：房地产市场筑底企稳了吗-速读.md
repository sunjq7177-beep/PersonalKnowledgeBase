# 房地产市场筑底企稳了吗——银河证券房地产高质量发展跟踪系列之一

## 一句话结论

一线城市（尤其上海）已出现筑底迹象，租金回报率自2025年4月起持续高于10年期国债收益率，房价跌幅收窄；但全国分化显著，地产板块处于低估值、低成交、低持仓区间，右侧机会仍需等待销售持续回暖与信用风险稳定。

## 最重要的边际变化

1. **上海价格端出现质变**：2026年2月上海二手住宅销售价格指数（上月=100）为100.2，是2025年5月以来首次超过100，即实现环比提升
2. **传导链条初现**：二手房销量回暖→有望向更多城市和新房扩散
3. **政策态度微调**：2026年政府工作报告从"持续用力推动房地产市场止跌回稳"变为"着力稳定房地产市场"，强调"探索多渠道盘活存量商品房"
4. **外资边际回暖**：地产板块外资流出有所收窄并出现边际回暖迹象

## 未来可能出现的现象

1. **一线城市二手房回暖向新房扩散**：传导链条为二手房销量回暖→新房销量回暖→房地产投资企稳→房价企稳
2. **"小阳春"行情延续**：2026年3月主要城市出现阶段性"小阳春"，一线及核心二线城市成交量改善
3. **政策加码路径**：因城施策→财政贴息/个税抵扣→地方政府收储（概率由大及小）
4. **租金回报率趋稳**：租金下行压力收窄，成交价格修复，整体租金回报率有望保持平稳

## 投资含义

**板块定位**：低估值、低成交、低持仓，行业最坏阶段或已过去

**看好标的**：
- 招商蛇口、保利发展、华润置地、华润万象生活、滨江集团、新城控股、招商积余、龙湖集团

**建议关注**：
- 优质开发：绿城中国、中国海外发展
- 优质物管：绿城服务
- 优质商业：恒隆地产
- 代建龙头：绿城管理控股
- 中介龙头：贝壳-W、我爱我家

**右侧机会条件**：销售持续回暖→房企信用风险进一步稳定→政策预期保持连贯→资金配置结构回补

## 作者主观观点压缩版

银河证券判断：**上海已初现筑底迹象**，核心依据是租金回报率已高于无风险收益率（10年期国债），且租金回报率正逐渐接近住房贷款利率。但区域分化是长期趋势，头部房企凭借低融资成本和核心区域高市占率优势，有望通过估值修复实现整体β机会。对于行业从底部走向阶段性行情，仍需观察销售回暖持续性、信用风险稳定、政策连贯性及资金配置回补等多条件逐步形成。
