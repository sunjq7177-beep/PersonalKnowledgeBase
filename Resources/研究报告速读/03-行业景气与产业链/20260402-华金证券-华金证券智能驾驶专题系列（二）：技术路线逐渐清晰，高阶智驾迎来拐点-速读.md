# 华金证券智能驾驶专题系列（二）：技术路线逐渐清晰，高阶智驾迎来拐点

## 一句话结论

智驾政策加速完善、L3/L4正式有法可依，城市NOA进入全面普及新周期，多家车企将城区NOA下放至10-20万车型，VLA与世界模型技术路线逐渐收敛，高阶智驾渗透率有望持续提升。

## 最重要的边际变化

1. **政策破冰**：2026年2月工信部发布首部L3/L4级自动驾驶强制国标征求意见稿，中国首批L3级车型（极狐阿尔法S6、长安深蓝SL03）2025年12月正式获得准入许可
2. **价格下沉**：城区NOA从30万+车型向10-20万车型扩散，比亚迪天神之眼C覆盖7万级车型，小鹏MONA M03 MAX仅13.98万带城市NOA
3. **技术收敛**：VLA与世界模型两条路线从竞争走向融合，理想MindVLA-o1、小鹏VLA 2.0、华为ADS 4均采用世界模型+端到端的融合架构
4. **算力跃升**：小鹏自研图灵芯片车端运行效率提升12倍，云端单GPU训练效率提升超1000%

## 图表速读

**图表1：城市NOA渗透率持续攀升**
- 2025年3-10月连续8个月渗透率超10%
- 2023年6.3%→2024年13.7%→2025年21.7%（新车标配率）
- 结论：城市NOA已过高端尝鲜期，进入规模化普及阶段

**图表2：2025年不同价格段城市NOA搭载量与渗透率**
- 25-30万渗透率57.0%，搭载量66.4万辆（最高）
- 20-25万渗透率29.7%
- 10-15万、15-20万开始起量，占比约10%
- 结论：高价车型渗透率趋近天花板，增量空间在20万以下

**图表3：2025年1-10月城市NOA第三方供应商市占率**
- Momenta：60.2%（搭载量34.8万辆）
- 华为HI：20.6%（搭载量11.9万辆）
- 其他（元戎启行、卓驭科技等）：19.2%
- 结论：Momenta绝对领先，华为HI增速可观

**图表4：智驾技术演进四阶段**
- 2004-2020年：模块化架构主导
- 2021-2023年：分段式端到端+规则算法融合
- 2024-2025年：多模态融合过渡期（端到端+VLM）
- 2026年-：超人智能全面落地（VLA+世界模型融合）

**图表5：VLA与世界模型融合路径**
- 路线1：世界模型做前置预测（最常见）
- 路线2：世界模型做后置校验/安全护栏
- 路线3：共享Transformer主干联合训练（前沿）

## 未来可能出现的现象

1. **2026年城市NOA渗透率有望突破20%**，全年搭载量冲击500万辆
2. **10-15万车型城市NOA将成为下一个爆发点**，比亚迪、吉利持续下探
3. **L3级自动驾驶2026年将有更多车型获准入许可**，从试点走向小规模商用
4. **VLA+世界模型融合方案将成为行业标配**，纯模仿学习路线逐渐被替代
5. **自研+外采混合模式成为多数车企选择**，全栈自研仅头部玩家能长期维持
6. **车企智驾差异化竞争加剧**，传感器配置（激光雷达数量）、城市覆盖数量、OTA频率成为核心差异点

## 投资含义

**主机厂**：关注智驾能力领先+强产品周期的小鹏汽车、小米集团、赛力斯、理想汽车、吉利汽车、比亚迪、零跑汽车

**零部件**：高阶智驾渗透率提升带动智能化零部件增量机会，建议关注：
- 千里科技（智驾方案）
- 电连技术（连接器）
- 拓普集团（底盘智能化）
- 德赛西威（域控制器）
- 伯特利（线控制动）
- 耐施特（智驾传感器）
- 经纬恒润（域控制器）

**核心逻辑**：城市NOA从高端选配向主流标配迁移，10-20万车型成为下一阶段主战场，技术路线收敛降低车企试错成本，智驾平权加速产业渗透率提升。

## 作者主观观点压缩版

黄程保（华金证券）认为：
1. 智驾行业已完成从"高端尝鲜"到"全面普及"的阶段跨越，2026年是高阶智驾规模化渗透的关键拐点
2. VLA与世界模型的高度互补性决定两者将深度融合而非替代，"超人智能"（超人类驾驶水平+超人类安全性）有望在2026年实现
3. 全栈自研的高投入决定仅特斯拉、华为、小鹏、理想等头部玩家可维持；自研+外采的混合模式将成为多数车企的主流选择
4. 法规从"有据可依"到"强制国标"，政策持续加码为智驾商业化扫清障碍
5. 第三方供应商（尤其Momenta）凭借规模量产和成本优势，将充分受益于智驾平权浪潮
