# 太平洋证券新能源+AI展望（第5期）：重视储能产业链调整带来的机会

## 一句话结论

储能产业链正从“鼓励建设”转向“改善回报”阶段，河南政策落地标志着地方储能支持从装机目标向收益机制深化，2026年继续看好格局好的储能环节（系统、隔膜、资源）及上游资源端业绩弹性。

## 核心数据卡

| 指标 | 数据 |
|------|------|
| 全国新型储能累计装机（2025年底） | 1.36亿千瓦/3.51亿千瓦时 |

| 河南2030年新型储能目标 | 23GW |
| 河南用户侧储能目标 | 8GW |
| 河南预计带动直接投资 | 400亿元 |

## 最重要的边际变化

1. **政策重心转移**：河南储能政策不再停留于装机目标，而是从收益机制入手——独立储能充电电量不承担输配电价和政府性基金附加，直接改善项目经济性
2. **储能定位升级**：新型储能正从前期的“政策培育型赛道”转向“新型电力系统关键基础设施”，《“十五五”规划纲要》明确提出“大力发展新型储能”
3. **收益机制破局**：政策推动储能由“成本项”加快向“收益型资产”演进，独立储能和用户侧储能项目经济性预期进一步清晰

## 未来可能出现的现象

1. 更多省份跟进河南模式，从“给指标”转向“给收益”
2. 独立储能项目IRR（内部收益率）预期改善，运营商资本开支意愿增强
3. 4小时及以上长时储能项目占比持续提升
4. 化工、钢铁、数据中心等高耗能场景成为用户侧储能新增量
5. 上游锂资源价格与储能需求形成正向联动，业绩弹性或超预期

## 投资含义

- **持续建议重视储能产业链**，尤其是格局好的环节：系统集成、隔膜、资源端
- **上游资源端业绩弹性**值得持续重视（赣锋锂业、天齐锂业、盐湖股份）
- 独立储能政策落地，利好储能系统集成商（阳光电源、德业股份）和电芯厂商（宁德时代）
- 中游材料端（天赐材料、湖南裕能、恩捷股份）受益于需求拉动

## 作者主观观点压缩版

分析师判断储能行业已进入“由规模扩张向价值兑现过渡”的新阶段。河南政策的边际意义在于：不再单纯追求装机数字，而是从电价减负、容量补偿、项目转型、重点场景落地四个维度实质性改善储能项目回报。作者认为这将推动储能产业链（尤其是格局好的系统、隔膜、资源环节）迎来估值与业绩的双重修复机会，上游资源端的业绩弹性或成为下一阶段超额收益来源。
