# 中国动力（600482）：船用发动机龙头，后市场+AIDC业务打开成长空间

## 一句话结论

全球船用发动机龙头，中船旗下核心动力装备平台，主业柴油机量价利齐升受益于造船景气延续，后市场服务与AIDC"船改燃"新业务打造第二、第三成长曲线。

## 最重要的边际变化

1. **利润增速迎向上拐点**：2025年前三季度归母净利润12.1亿元，同比+62%，毛利率从2022年12.8%提升至2025Q1-3的16.0%
2. **新造船在手订单创历史新高**：628亿元，较年初增长7%，头部船厂新船交付排至2030年
3. **中低速机全球份额稳固**：2025年中船集团全球船用低速机份额约43%，中国/韩国/日本份额52%/35%/13%
4. **双燃料技术升级**：替代能源占新签订单27%（2024年33%），发动机价值量显著提升
5. **AIDC需求爆发**：燃气轮机订单排至2029年，"船改燃"成为补充方案，中国动力具备技术储备

## 未来可能出现的现象

1. **造船交付高峰**：克拉克森预计2026-2027年全球船舶交付量5030/5600万CGT，同比+14%/+11%
2. **低速机后市场放量**：公司已建立5（国内）+3（国外）服务网络，后市场具备周期性弱、盈利水平高的特点
3. **AIDC"船改燃"规模化**：2025年瓦锡兰已签789MW数据中心订单，2027年交付；中国动力中速机与双燃料技术储备深厚，具备船改燃能力
4. **少数股权收购重启**：公司定位为中船集团动力总成平台，若收购中船柴油机16.5%股权完成，归母净利润有望增厚10亿元以上
5. **全球船队更新加速**：约1/3船舶船龄超15年，2026-2030年全球年均交付需求约1.1亿DWT，更新需求占比超50%

## 投资含义

1. **核心看点**：公司为全球船用发动机龙头，柴油机量价利齐升受益造船景气延续，双燃料技术升级提升价值量
2. **第二成长曲线**：后市场服务网络初步建成，具备高盈利、弱周期特点
3. **第三成长曲线**：AIDC缺电带动燃气轮机/内燃机需求爆发，"船改燃"打开新空间
4. **估值安全垫**：2025-2027年PE 33/23/18倍，首次覆盖给予"买入"评级
5. **风险提示**：国际贸易政策变化、原材料价格波动、汇率波动

## 作者主观观点压缩版

**为什么这是核心资产？**
中国动力占据全球船舶动力系统研发设计、集成制造、设备配套、保障服务龙头地位，中船集团持股54.67%，定位为集团动力总成平台。柴油机业务控股子公司中船柴油机52%股权，若后续完成16.5%少数股权收购，归母净利润有望增厚10亿元以上。2025H1新签订单339亿元，在手订单628亿元创历史新高，订单交付排至2030年。

**为什么现在有机会？**
①造船供需缺口短中期难以消解：全球船队平均船龄13.4年（高于2013年低点9.7年），约1/3船舶船龄超15年，2026-2030年均交付需求1.1亿DWT；②全球活跃船厂数较2010年下降45%，产能扩张缓慢；③双燃料技术升级驱动价值量提升，替代燃料占新签订单27%；④AIDC爆发带动燃气轮机/内燃机需求，"船改燃"成为补充方案，中国动力在船用中速机、燃气轮机领域技术储备深厚。

**盈利预测斜率怎么看？**
预计2025-2027年归母净利润22/31/41亿元，同比+55%/+44%/+32%。增量来自：①造船高景气下柴油机板块量价利齐升；②后市场服务放量；③AIDC"船改燃"贡献增量。当前市值对应PE 33/23/18倍，具备估值吸引力。
