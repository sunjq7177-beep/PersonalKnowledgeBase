# 恒铭达（002947）：消费电子基本盘稳固，AI与新能源打造第二成长极

## 一句话结论

精密制造龙头深度绑定苹果供应链，受益"AI+轻薄"重塑消费电子主业，同时通过AI算力底座与新能源充电桩构筑第二成长极，惠州基地2026年投产驱动业绩高弹性。

## 最重要的边际变化

1. **AI眼镜新品弹性**：预计2026年苹果AI眼镜首年销量705万台，公司特种胶带、精密结构胶、声学功能件价值量提升
2. **iPhone 17 Air去螺丝化**：轻薄化趋势带动精密粘贴固定类器件单机用量提升，屏蔽类功能器件受益高频高速需求
3. **华为AI超节点落地**：2025年Atlas 900 A3已出货300台，驱动精密结构件需求倍增
4. **惠州基地2026年投产**：精准承接国产智算爆发期订单，强化大尺寸精密金属结构件加工能力
5. **小鹏汽车充电桩配套**：预计2026年公用超充桩新增需求3.6万座，私桩需求15.4万座

## 未来可能出现的现象

1. **苹果新品周期**：2026年iPhone 17 Air/折叠屏驱动精密胶粘件用量提升；AI眼镜705万台首年出货
2. **华为算力集群扩张**：Atlas 950/960超节点逐步落地，2026Q4支持8192颗Ascend芯片
3. **车桩比优化**：工信部"1:1"目标指引下，小鹏公桩超充站2026年底达4500座
4. **惠州基地产能释放**：2026年建成后承接AI算力与新能源订单，精密金属结构件毛利率边际改善

## 投资含义

1. **盈利预测斜率**：2025→2028年收入CAGR约35%，归母净利润CAGR约31%
2. **估值框架**：2026年PE 18x，显著低于可比公司均值22x；PB 3.52x处于历史中枢
3. **增长驱动力**：消费电子基本盘稳固（苹果链）+ 通信业务受益华为AI超节点 + 新能源绑定小鹏充电桩
4. **核心风险**：消费电子需求波动、惠州基地投放节奏不及预期、新业务放量低于预期

## 作者主观观点压缩版

恒铭达是典型的"苹果链精密制造龙头"转型样本。公司核心优势在于"材料-工艺-制造"全链条协同，已从单一防护件向精密功能件升级。AI新周期带来三重机遇：

1. **消费电子主业升级**：AI终端轻薄化+高频化→精密胶粘件+屏蔽件价值量双升
2. **AI算力底座卡位**：华为超节点大规模Scale-up路线→需要更多物理机柜与交换机结构件
3. **新能源充电桩**：绑定小鹏汽车，"私桩+公桩超充"双轮驱动

惠州基地2026年投产是关键产能节点，届时将精准承接国产智算+充电网络建设需求。短期估值显著低于可比公司，中长期成长逻辑清晰，具备较高安全边际。
