# 河南新型储能政策核心措施

![[Resources/投研数据库/00-图表证据/行业/储能/02-河南新型储能政策核心措施-p01.png|1000]]

## 图表说明

河南储能政策从"给指标"转向"给收益"，政策核心措施一览。

## 核心数据

| 措施类型 | 具体内容 |
|----------|----------|
| 电价减负 | 独立储能充电电量不承担输配电价及政府性基金附加 |
| 容量补偿 | 健全容量电价机制 |
| 项目转型 | 支持新能源配建储能按程序转为独立储能 |
| 场景拓展 | 鼓励化工、钢铁、数据中心配置用户侧长时储能 |

## 政策目标

- 河南2030年新型储能目标：23GW
- 河南用户侧储能目标：8GW
- 预计带动直接投资：400亿元

## 投资含义

河南政策破局标志着储能支持政策从装机目标向收益机制的深化，是储能项目经济性改善的关键信号，对储能系统集成商（阳光电源、海博思创）和独立储能运营商直接利好。
