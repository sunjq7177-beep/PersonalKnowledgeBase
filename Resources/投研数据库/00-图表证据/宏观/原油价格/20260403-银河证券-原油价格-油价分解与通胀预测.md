# 原油价格-油价分解与通胀预测

![[20260403-银河证券-原油价格-油价分解与通胀预测-p04.png|1000]]

## 图表一句话

这张候选页截图把这轮油价上行拆成了几个明确来源，并把不同油价水平对应的 CPI / PPI 路径直接画出来，适合拿来做“原油价格卡”的上层证据。

## 这张图在说什么

- 页上方的表格把油价中枢抬升拆成五部分，合计把中枢从 `60-70` 美元抬升到 `75-85` 美元。
- 左下图给出不同油价情景下 CPI 同比的预测路径，说明油价在 `85 / 100 / 120 / 150` 美元时，通胀抬升幅度明显分层。
- 右下图给出不同油价情景下 PPI 同比的预测路径，PPI 对油价冲击更敏感，弹性明显高于 CPI。
- 这不是单纯在说“油价涨了”，而是在说油价上行已经足以改变未来几个月的通胀斜率。

## 为什么保留它

- 它把“油价上涨”从叙事拆成了可追踪的几项来源。
- 它同时给出 CPI / PPI 预测，适合后续挂接 `[[PPI与通胀压力]]`。
- 对投资最有用的是，它把 `85 美元` 这个温和通胀区间和更高油价情景直接区分出来了。

## 它支持或削弱了什么判断

- 支持：本轮油价不是纯粹脉冲，更像中枢抬升。
- 支持：如果油价稳定在 `75-85` 美元，国内通胀大概率是“温和回升”，不是立刻失控。
- 支持：PPI 比 CPI 更先、也更明显地反映油价冲击。
- 削弱：把这轮冲击简单当成一次性噪音的判断。

## 最该看的点

- 重点看页上方表格最后一行“合计中枢抬升”。
- 重点看左下图里 `预测: 油价85` 与 `预测: 油价120/150` 的斜率差异。
- 重点看右下图里 PPI 在高油价情景下的弹性放大。
