# 美元指数：短强长弱，定价之锚松动

## 当前一句话判断

美元短期因流动性优势+避险需求维持强势，但财政利息支出膨胀、通胀黏性、去美元化加速正在侵蚀中长期信用根基，中长期弱势难以逆转。

## 当前边际变化

- **强化**：美伊冲突持续，霍尔木兹通航受限，避险资金短期流向美元
- **弱化1**：财政利息支出逼近1万亿美元/年，信用重估压力加大
- **弱化2**：通胀黏性超预期，实际利率下行压制美元信用
- **弱化3**：黄金+原油同步上涨，暗示美元体系承压

## 关键阈值与失效条件

| 阈值/事件 | 含义 |
|----------|------|
| 特朗普TACO兑现 | 美元中期顶部确认，长弱逻辑加速 |
| 10年期美债收益率突破前高 | 通胀压过避险，美元短强难持续 |
| 霍尔木兹恢复通航 | 避险消退，美元支撑快速瓦解 |
| 美债大规模抛售 | 美元信用危机触发 |
| 美国经济超预期衰退 | 衰退交易主导，美元走弱 |

**失效条件（美元重新走强）**：地缘冲突超预期升级 / 美联储超预期鹰派加息 / 全球避险资金大规模流入

## 对投资的直接指导

1. **看什么**：美元趋势性走弱确认时，优先配有色金属（铜、铝），弹性优于黄金
2. **配什么**：新兴市场资产边际改善，中国远离冲突核心区，政策空间充足
3. **少做什么**：黄金短期承压，等待更好买点；避免在美元强势确认阶段追高
