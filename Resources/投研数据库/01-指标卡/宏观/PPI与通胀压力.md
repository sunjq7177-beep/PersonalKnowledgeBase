# PPI与通胀压力

## 当前一句话判断

PPI 3月转正至+0.2%，但80美元/桶滞胀边界已破，通胀传导从预期变为现实。

## 当前边际变化

- PPI 3月转正，PMI出厂价格与原材料购进价格双双大幅上行，通胀传导已在数据端确认
- **80美元/桶滞胀边界已破**，当前104美元，PPI向上空间打开
- PMI原材料购进价格63.9创本轮新高，成本压力向下游传导加速
- 银河证券预测模型：85美元→PPI约2%，100美元更高，120美元以上持续上行

## 关键阈值与失效条件

| 阈值 | 含义 |
|------|------|
| **80美元/桶** | **滞胀风险边界，当前已破（104美元）** |
| PPI转正 | 工业通缩压力缓解（已触发） |
| 100美元/桶 | 持续性通胀风险确认 |
| CPI 3% | 通胀警戒线（美国） |

**失效条件**：霍尔木兹海峡恢复通航/OPEC紧急增产→油价回落至75美元以下；全球需求超预期崩塌（衰退交易主导）。

## 对投资的直接指导

1. **看多**：PPI转正→A股非金融企业盈利回升预期增强，重点关注煤化工、新能源替代方向
2. **看空**：油价高位挤压中下游利润，消费和出口链承压；美联储若被迫收紧，美股估值承压
3. **配置**：滞胀防御（必选消费、黄金、大宗商品）+国内能源替代和储能逻辑
