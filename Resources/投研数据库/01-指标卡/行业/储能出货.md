# 储能出货

## 当前一句话判断

**储能产业链由"鼓励建设"转向"改善回报"，招标数据持续高增长（3月环+49%），河南模式破局推动独立储能IRR改善，中长期看好，短期Q4毛利率压力需跟踪。**

## 当前边际变化

1. **河南政策破局**：独立储能充电电量不承担输配电价及政府性基金附加，直接改善项目经济性，是从"装机目标"向"收益机制"转变的标志
2. **招标数据持续强劲**：3月环增49%验证下游需求旺盛，排产持续改善（4月+5%，5月+10%）
3. **美国大储高增**：2026年预计80GWh（+51%），数据中心贡献37GWh成为边际增量
4. **格局好的环节（系统、隔膜、资源）业绩弹性显现**

## 关键阈值 / 失效条件

| 条件类型 | 具体内容 |
|----------|----------|
| **失效条件1** | 招标数据环比连续2个月下降 → 需求拐头 |
| **失效条件2** | 碳酸锂价格快速下跌（<12万/吨）→ 资源端业绩弹性消失 |
| **失效条件3** | 配储时长小时数下降 → 项目经济性恶化 |
| **关键阈值** | 全年招标能否突破200GWh（2025年约150GWh） |
| **关键阈值** | 河南模式能否被更多省份跟进 |

## 对投资的直接指导

**当前判断**：看多（高置信度）

**最直接受益方向**：
1. **系统集成商**（阳光电源、海博思创）：规模扩张+格局优化
2. **隔膜**（恩捷股份）：格局好，盈利韧性强
3. **上游锂资源**（赣锋锂业、天齐锂业）：业绩弹性来源
4. **北美大储供应链**（阿特斯、德业股份）：美国市场高增

**操作建议**：当前处于储能产业链"由规模扩张向价值兑现过渡"阶段，逐步布局格局好的环节
