# 智驾渗透率

## 当前一句话判断

**城市 NOA 已经过了“高端尝鲜”阶段，进入规模化渗透早中段。真正值得盯的边际变化，不是 30 万以上继续卷，而是 `20 万以下开始起量`。**

## 当前边际变化

1. **渗透率连续抬升**：不是单月跳点，而是连续多个月加速。
2. **价格带下沉是真正的新变量**：高价带已经验证，低价带起量才决定行业天花板。
3. **政策+技术路线同时顺风**：L3/L4 规则推进，VLA/世界模型融合降低车企试错成本。
4. **第三方方案商地位变重要**：如果车企不全栈自研，供应链弹性会更大。

## 关键阈值 / 失效条件

| 阈值 | 含义 |
|------|------|
| **20%+** | 新车城市 NOA 标配率跨过规模化临界点 |
| **500万辆** | 2026 年全年搭载量冲击目标 |
| **20万以下持续起量** | 真正决定行业扩圈速度 |

**失效条件：**
- 10-15 万、15-20 万价格带起量不及预期
- L3/L4 国标和上路节奏明显延后
- 头部方案商份额快速失守，行业竞争恶化到价格战

## 对投资的直接指导

1. **优先看“有量”的环节**：渗透率抬升后，最先兑现的是整车销量和 Tier1 配套量。
2. **别只盯高端整车**：真正的增量在 10-20 万价位，第三方方案商、域控、线控制动、连接器更有弹性。
3. **关注价格带验证，而不是口号**：后面每一次销量和标配率更新，都要看低价带有没有继续扩大。
