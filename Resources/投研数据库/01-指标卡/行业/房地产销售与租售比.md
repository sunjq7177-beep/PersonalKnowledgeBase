# 房地产销售与租售比

## 当前一句话判断

**上海租金回报率首破2%（2016年来首次>10年期国债），核心城市筑底信号出现，但70城房价同比仍负、三线仍有流拍，全国性右侧机会需等待销售持续回暖验证。**

## 当前边际变化

1. **上海价格端出现质变**：2026年2月上海二手住宅销售价格指数环比+0.2%，是2025年5月以来首次>100
2. **租金回报率突破关键阈值**：2025年12月达2.01%，已持续高于10年期国债收益率
3. **传导链条初现**：二手房销量回暖→有望向更多城市和新房扩散
4. **政策态度微调**：2026年政府工作报告从"持续用力推动房地产市场止跌回稳"变为"着力稳定房地产市场"

## 关键阈值 / 失效条件

| 条件类型 | 具体内容 |
|----------|----------|
| **失效条件1** | 上海租金回报率再次跌破10年期国债收益率 → 筑底信号失败 |
| **失效条件2** | 70城房价指数同比负值持续扩大 → 全面复苏推迟 |
| **失效条件3** | 房企信用风险再度爆发（如头部房企违约）→ 行业右侧延后 |
| **关键阈值** | 上海二手房月均成交能否持续>100万方（2026年1-2月已达125万方） |
| **关键阈值** | 70城房价环比何时转正（2026年2月上海已出现） |

## 对投资的直接指导

**当前判断**：中性（底部区域信号出现）

**看好标的（具备相对收益）**：
- 招商蛇口、保利发展、华润置地
- 滨江集团、新城控股
- 招商积余、龙湖集团
- 贝壳-W（中介龙头）

**右侧机会条件**（需全部满足）：
1. 销售持续回暖（重点跟踪周度成交数据）
2. 房企信用风险进一步稳定
3. 政策预期保持连贯
4. 资金配置结构回补

**操作建议**：一线城市已出现筑底信号，可逐步关注优质开发商，但全国性右侧机会仍需等待验证
