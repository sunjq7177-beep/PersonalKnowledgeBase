# DR007

## 当前一句话判断

**DR007 还在极松区，但继续更松的赔率已经不高。现在最重要的不是它有没有马上上到 `2%`，而是它会不会从 `1.4%` 一线缓慢回到中性，从而把短端拥挤挤掉。**

## 当前边际变化

1. **极松还在，但边际收口信号已经出现**：逆回购投放压到 `5 亿元`，买断式回购净回笼已在做。
2. **资金利率低，不等于短端有大机会**：现在更像“很松，但太贵”，不是“很松，还能继续下”。
3. **4 月供给与缴款是下一道坎**：政府债净缴款、特别国债发行，都可能把 DR007 往上推一点。
4. **真正该防的是回归中性，不是瞬间收紧**：这对短端交易影响更大，因为赔率已经被吃薄。

## 关键阈值 / 失效条件

| 阈值 | 含义 |
|------|------|
| **1.30%-1.40%** | 当前极松区 |
| **1.50%** | 开始脱离极松、向中性回归 |
| **2.00%** | 明显收紧区，短端压力会显著上升 |

**失效条件：**
- 央行重新明显加大投放，DR007 长时间维持超低位
- 政府债供给低于预期，资金面扰动消失
- 经济再度转弱，政策被迫重新加码宽松

## 对投资的直接指导

1. **短端别追**：现在更像止盈区，不像再加仓区。
2. **杠杆别再激进**：资金便宜不代表杠杆仍然划算，历史高杠杆本身就放大波动。
3. **看空短端要等确认**：真要做空，也更适合等 DR007 站上 `1.5%` 再做。
4. **长端交易也得看它脸色**：DR007 一旦回归中性，长端下行空间会先被短端传导压缩。
