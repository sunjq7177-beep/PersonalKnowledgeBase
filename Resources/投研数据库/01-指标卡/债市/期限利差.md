# 期限利差（10Y-1Y）

## 当前一句话判断

**`60bp` 已经是高位区，不是舒服的继续做陡位置。现在更合理的默认方向，是等它往 `50bp` 甚至更低压，而不是继续赌它无限扩。**

## 当前边际变化

1. **曲线已经够陡**：从 1Y 到 10Y 的斜率很明显，市场已经把“短端松、长端受压”交易得比较充分。
2. **继续走扩需要新利空**：比如 PMI/PPI 同步继续往上、供给压力进一步抬升。
3. **压缩触发条件反而更常见**：资金面回归中性、风险偏好修复、长端重新找配置买盘，都可能推平曲线。

## 关键阈值 / 失效条件

| 阈值 | 含义 |
|------|------|
| **40bp** | 偏平区，长端更强时常见 |
| **50bp** | 中性区 |
| **60bp** | 当前高位区 |
| **68bp** | 继续做陡前要特别谨慎的上沿 |

**失效条件：**
- PMI / PPI 再超预期上行，长端被继续打压
- 央行明显放松，短端重新更低，曲线再被动走陡
- 地缘冲突升级，短期避险买盘重新集中到长端

## 对投资的直接指导

1. **做陡不划算了**：现在再买 10Y 空 1Y 的性价比不高。
2. **更像等做平窗口**：如果你要做曲线，方向上更偏向等利差压缩。
3. **看长端也别太乐观**：长端不是没风险，而是已经不便宜。
4. **跨品种替代更舒服**：5-10 年国开债比直接硬做 10Y 国债更顺手。
