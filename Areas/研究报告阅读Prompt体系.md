# 研究报告快速阅读 Prompt 体系

## 先说结论

- 这 40 篇报告，按内容主体可以稳定归成 5 类：
  - 宏观与大类资产
  - 利率债与债市交易
  - 行业景气与产业链
  - 公司/个股与财报估值
  - 市场主题与交易复盘
- 真正要提取的主线不是“作者说了什么”，而是：
  - 数据现在是什么
  - 数据相对上期怎么变了
  - 边际变化在验证什么、证伪什么
  - 这些变化后面可能推演出什么现象
  - 对投资动作到底意味着什么
- 图表不能当配角。很多报告真正有价值的信息，不在摘要段落里，而在图表和表格里。

---

## 为什么最终是这 5 类

| 类别 | 样本数量 | 典型样本 | 你真正要抓的东西 |
|---|---:|---|---|
| 宏观与大类资产 | 16 | PMI 数据点评、二季度大类资产配置、策略周报、原油择时 | 宏观变量、资产联动、配置方向 |
| 利率债与债市交易 | 3 | 债市回顾与展望、固收周报、长短端判断 | 收益率曲线、资金面、供给与交易拥挤度 |
| 行业景气与产业链 | 15 | 新能源+AI、算力、智能驾驶、光模块、房地产、铁矿石 | 需求、供给、价格、库存、渗透率、资本开支 |
| 公司/个股与财报估值 | 4 | 恒铭达、中国动力、豪威、阳光电源财报点评 | 营收利润、毛利率、现金流、订单、估值空间 |
| 市场主题与交易复盘 | 2 | 主题交易复盘、短期见底了吗 | 成交、情绪、风格、板块轮动、催化节奏 |

## 先判内容主体，不迷信文件夹

- 文件夹只是粗分类，真正阅读时要按正文主体选 Prompt。
- 例如：
  - `阳光电源` 这几篇虽然放在行业目录里，但正文核心是财报、利润率、储能出货、估值，优先走“公司/个股与财报估值”。
  - `证券行业 2025 年年报总结` 虽放在个股目录里，但本质是行业经营数据汇总，优先走“行业景气与产业链”。
- 判断标准很简单：
  - 主体是宏观变量/资产配置：宏观与大类资产
  - 主体是收益率/资金面/利差：利率债与债市交易
  - 主体是行业供需/价格/出货/渗透率：行业景气与产业链
  - 主体是单公司财报/订单/估值：公司/个股与财报估值
  - 主体是市场热度/主题轮动/周度复盘：市场主题与交易复盘

## 统一快速阅读总框架

### 1. 先找数据，不先找观点

先把下面四类信息抄出来，再谈结论：

| 必抄项 | 你要写什么 |
|---|---|
| 当前值 | 现在是多少 |
| 对比值 | 上月、上季、去年同期、市场预期、历史均值 |
| 变化方向 | 上升/下降、加速/减速、改善/恶化 |
| 投资含义 | 这意味着什么资产、行业、公司受益或受损 |

### 2. 边际变化优先级

优先抓这 4 种变化：

1. 从差变好，或者从好变差
2. 变化幅度超预期
3. 图表出现拐点、背离、二阶导变化
4. 一个数据变化开始传导到第二层变量

### 3. 图表速读固定动作

每张值得看的图表，都按这个顺序写：

1. `图/表编号 + 标题`
2. `关键变量 + 单位`
3. `当前值/近期高低点/同比环比`
4. `趋势：延续、拐点、背离、验证还是证伪`
5. `对投资判断的含义`

### 4. 统一输出结构

以后无论哪类报告，都尽量逼 AI 按这个顺序输出：

1. 一句话结论
2. 核心数据卡
3. 最重要的边际变化
4. 图表速读
5. 未来可能出现的现象
6. 投资含义
7. 作者主观观点压缩版

### 5. 硬规则

- 所有数字都尽量写 `当前值 + 对比值 + 变化`
- 图表和表格优先级高于段落抒情
- 主观观点只保留 1-2 句
- 明确区分：
  - `事实数据`
  - `作者判断`
  - `基于数据的推演`

## 1. 宏观与大类资产 Prompt

### 适用样本

- PMI 数据点评
- 宏观月报 / 季报 / 周报
- 大类资产配置报告
- 原油、美元、美债、黄金联动类报告
- 地缘冲突对资产价格影响的策略报告

### 先看什么数据

- 经济景气：GDP、PMI、生产、新订单、出口、就业
- 通胀与盈利：CPI、PPI、工业利润
- 流动性与利率：中美 10Y、期限利差、降息预期、美元
- 风险资产与避险资产：原油、黄金、美股、A 股、港股
- 配置结论：超配/标配/低配什么

### 最重要的边际变化

- PMI 是否重新站上 50，分项里谁先改善
- 原油、运价、美元、美债收益率是否开始联动放大
- 通胀预期与衰退预期，谁在主导交易
- 同一套宏观冲击下，股票、债券、商品谁强谁弱

### 图表怎么读

- 大类资产涨跌幅表：先看相对强弱，再看是否扩散
- PMI 分项图：看生产、新订单、原材料、就业谁先变
- 原油/黄金/美债/美元联动图：看冲击是通胀型还是衰退型
- 地缘或政策事件时间轴：看风险是一次性冲击还是持续发酵

### 如何把数据转成投资判断

- 增长回升 + 通胀温和：顺周期、权益更友好
- 原油上行 + 利率上行 + 风险资产承压：更像通胀冲击
- 原油上行但需求走弱、收益率曲线收窄：更像衰退交易
- 同时看“绝对值”和“变化方向”，不要只看一个点位

```markdown
请严格按下面结构输出：

1. 一句话结论
- 用 30 字内写出这篇报告最重要的宏观交易信号。

2. 核心数据卡
| 指标 | 当前值 | 对比值/预期 | 变化 | 对投资的含义 |
|---|---:|---:|---|---|
| GDP/PMI |  |  |  |  |
| CPI/PPI |  |  |  |  |
| 出口/社融/就业 |  |  |  |  |
| 中美 10Y / 利差 |  |  |  |  |
| 原油 / 黄金 / 美元 |  |  |  |  |
| 大类资产配置建议 |  |  |  |  |

3. 最重要的边际变化
- 只列 3-5 条。
- 每条必须写清：
  - 什么数据变了
  - 是同比/环比/较上期还是超预期
  - 这在验证什么交易逻辑

4. 图表速读
- 选最重要的 3-6 张图/表。
- 每张按这个格式写：
  - 图/表编号 + 标题
  - 关键变量与单位
  - 趋势 / 拐点 / 背离
  - 对资产价格的含义

5. 未来可能出现的现象
- 基于数据，不要空谈。
- 只写 2-3 条最可能发生的传导：
  - 例如：原油继续上行 -> 通胀预期抬升 -> 美债利率易上难下 -> 成长股承压

6. 投资含义
- 利多什么
- 利空什么
- 接下来最该跟踪哪 3 个阈值或数据

7. 作者主观观点压缩版
- 只保留 1-2 句。
- 单独标注这是“作者观点”，不要和事实数据混在一起。

规则：
- 所有数字尽量写“当前值 + 对比值 + 变化”。
- 如果摘要和图表冲突，以图表和表格为准。
- 不要大段复述背景故事。
- 明确区分“事实数据”“作者判断”“基于数据的推演”。
```

## 2. 利率债与债市交易 Prompt

### 适用样本

- 债市月报 / 周报
- 跨季资金面跟踪
- 收益率曲线与利差分析
- 长端空间 / 短端拥挤类报告

### 先看什么数据

- 国债/国开债各期限收益率
- 10Y-1Y、10Y-3Y 等期限利差
- DR007、R007、同业存单、OMO/MLF
- 政府债供给、信用债发行、取消发行
- 配置盘/交易盘行为、久期与杠杆

### 最重要的边际变化

- 曲线是变陡还是变平
- 短端是不是拥挤，长端是不是开始有性价比
- 跨季之后资金价格是否回落
- 利多债市的是基本面走弱、还是流动性改善
- 供给冲击有没有改变长端定价

### 图表怎么读

- 收益率曲线图：看绝对利率和曲线形态
- 资金利率图：看是否持续高于政策利率
- 发行与净融资图：看供给压力是否上来
- 机构行为图：看谁在买、谁在撤

### 如何把数据转成投资判断

- 基本面偏弱 + 资金面友好：债券更占优
- 供给冲击 + 短端拥挤：短久期未必舒服，长端看节奏
- 如果“逻辑没变但交易拥挤”，就要防止赔率下降

### 可直接复制的 Prompt

```markdown
请把这篇报告按“利率债与债市交易速读模式”拆解，只保留对交易有用的内容。

请严格按下面结构输出：

1. 一句话结论
- 这篇报告到底在说：债市偏多、偏空，还是逻辑未定？

2. 核心数据卡
| 模块 | 当前值 | 对比值 | 变化 | 含义 |
|---|---:|---:|---|---|
| 1Y/3Y/5Y/10Y 国债收益率 |  |  |  |  |
| 1Y/3Y/5Y/10Y 国开债收益率 |  |  |  |  |
| 期限利差 |  |  |  |  |
| DR007 / R007 / 同业存单 |  |  |  |  |
| 政府债供给 / 信用债发行 |  |  |  |  |
| 久期 / 杠杆 / 机构行为 |  |  |  |  |

3. 最重要的边际变化
- 只列 3-5 条：
  - 收益率变动发生在哪个期限
  - 资金面是更松还是更紧
  - 供给与交易拥挤度有没有变化

4. 图表速读
- 选最关键的 3-6 张图/表。
- 每张都写：
  - 图/表编号 + 标题
  - 关键变量
  - 当前曲线或利率位置
  - 边际变化
  - 对久期/杠杆/仓位的含义

5. 未来可能出现的现象
- 只写 2-3 条最重要的推演：
  - 例如：跨季后资金回落 -> 短端修复
  - 例如：供给上量 -> 长端承压

6. 投资含义
- 当前更适合做什么：
  - 拉久期 / 缩久期 / 票息 / 等待
- 当前最该盯的 3 个数据

7. 作者主观观点压缩版
- 1-2 句即可。

规则：
- 变化统一尽量用 BP 表示。
- 必须区分“基本面逻辑”“资金面逻辑”“供给逻辑”“交易拥挤逻辑”。
- 不要只说多空，要把原因拆开。
```

## 3. 行业景气与产业链 Prompt

### 适用样本

- 行业周报 / 月报 / 深度
- 新能源、算力、智能驾驶、光模块、房地产、大宗商品
- 以行业景气、供需、价格、出货、渗透率为核心的报告

### 先看什么数据

- 需求：销量、出货、装机、订单、招标
- 供给：产能、开工率、资本开支、扩产节奏
- 价格：现货价、合约价、 ASP、吨价、单瓦盈利
- 库存：产业链库存、渠道库存、去库/补库
- 结构：渗透率、份额、技术升级路径

### 最重要的边际变化

- 需求是在恢复、放缓，还是超预期
- 价格是在止跌、加速下跌，还是开始回升
- 库存是在继续去化，还是又开始累积
- 龙头扩产/资本开支，是景气验证还是供给过剩前兆
- 主题是不是从“讲故事”转成“有数据验证”

### 图表怎么读

- 出货/装机图：看量的方向和弹性
- 价格曲线：看是否见底或出现拐点
- 库存图：看去库是否进入后半段
- 渗透率/份额图：看成长逻辑还剩多少空间
- 资本开支/扩产图：看未来供给压力

### 如何把数据转成投资判断

- 量升 + 价稳/价升：最强
- 量升 + 价跌：更偏交易机会，要防盈利兑现
- 去库尾声 + 价格企稳：往往是赔率改善点
- 高景气但扩产过快：要警惕下一阶段供给过剩

```markdown
1. 一句话结论
- 这个行业现在最关键的判断是什么：景气上行、见底修复、分化加剧，还是故事多于数据？
```

## 4. 公司/个股与财报估值 Prompt

### 适用样本

- 个股深度
- 年报 / 季报点评
- 目标价 / 估值调整
- 订单、现金流、第二成长曲线类公司报告

### 先看什么数据

- 营收、归母净利润、EPS
- 毛利率、净利率、费用率、经营现金流
- 分业务数据：哪些板块拖累、哪些板块增长
- 订单、在手、出货、产能、资本开支
- 估值：PE、PB、目标价、可比公司

### 最重要的边际变化

- 单季收入和利润是加速还是放缓
- 毛利率是阶段性承压还是趋势性恶化
- 现金流是否明显强于或弱于利润
- 订单和在手，是否支持未来几个季度
- 目标价下调/上调，核心是盈利预测变了还是估值倍数变了

### 图表怎么读

- 财务预测表：看 2-3 年盈利斜率
- 分业务收入利润图：看第二增长曲线是否真实
- 订单/在手图：看业绩持续性
- 可比估值图：看当前溢价/折价来自哪里

### 如何把数据转成投资判断

- 业绩强 + 现金流强 + 订单强：质量高
- 收入增但利润弱：先看毛利率和费用率
- 估值空间来自盈利上修还是估值切换，要分开看
- 长逻辑没坏但单季承压，要判断是短期扰动还是拐点转弱

```markdown
  - 单季变化

  - 关键变量
  - 当前趋势 / 拐点
  - 对盈利、估值或兑现度的含义

5. 未来可能出现的现象
- 只写 2-3 条：
  - 例如：储能高增长延续 -> 盈利结构改善
  - 例如：订单放缓 -> 下一阶段收入斜率下修

6. 投资含义
- 当前属于：
  - 业绩兑现型
  - 估值修复型
  - 第二成长曲线验证型
  - 逻辑未证实型
- 最该继续跟踪哪 3 个指标

7. 作者主观观点压缩版
- 评级、目标价、核心假设用 1-2 句带过。

规则：
- 一定区分“公司业绩变化”和“估值倍数变化”。
- 如果目标价变了，要拆清楚到底是 EPS 变了还是 PE 变了。
- 如果报告里有大量图表，优先读财务预测表、分业务图、订单图、估值对比图。
```

## 5. 市场主题与交易复盘 Prompt

### 适用样本

- 每周主题交易复盘
- 市场见底/风格切换/热点扩散类报告
- 板块轮动、情绪修复、下周展望类报告

### 先看什么数据

- 指数表现、成交额、融资、涨跌停、市场宽度
- 强势/弱势行业与主题
- 龙头股、扩散股、补涨股
- 催化时间点：政策、业绩、产业事件
- 情绪位置：冰点、修复、高潮、分化

### 最重要的边际变化

- 成交额是否放大或缩量
- 热点是集中还是扩散
- 龙头是否继续超额，还是只剩尾部补涨
- 情绪是从悲观修复，还是从拥挤转分化
- 当前主题是基本面验证，还是纯交易情绪

### 图表怎么读

- 指数与成交额图：看市场承接力
- 板块涨跌幅表：看主线是否集中
- 情绪指标图：看修复还是退潮
- 事件时间轴：看主题还能不能继续发酵

### 如何把数据转成投资判断

- 放量 + 主线集中 + 龙头强化：趋势延续概率更高
- 缩量 + 补涨扩散 + 龙头走弱：更像后半段
- 市场见底类报告，重点不是结论，而是“见底条件是否真的被数据满足”

## 3-5 分钟读完一篇报告的极简流程

1. 先判类型
- 先判断它到底属于哪 1 类，不要被文件夹误导。

2. 先看图表目录和摘要
- 找 3-6 张最重要的图/表。

3. 抄核心数据
- 只抄当前值、对比值、变化，不抄废话。

4. 找边际变化
- 问自己一句：这篇报告真正新增的信息是什么？

5. 写未来现象
- 基于数据，推 2-3 条最可能发生的传导。

6. 落到投资动作
- 利多什么、利空什么、还要继续盯什么。

## 使用建议

- 默认先用“统一快速阅读总框架”。
- 明确类型后，再套对应的 5 类 Prompt。
- 混合型报告处理原则：
  - 主体是行业数据，就用行业 Prompt
  - 主体是公司财报，就用公司 Prompt
  - 主体是市场热度，就用主题 Prompt
- 如果你后面要继续细化，我最建议先拆这两个方向：
  - `行业景气与产业链`：再拆成“成长赛道”与“周期品”
  - `公司/个股与财报估值`：再拆成“财报点评”与“深度覆盖”
