# 未来90天交易执行计划

## 总目标

未来 90 天的目标不是“立刻稳定盈利”，而是先做到三件事：

1. 交易类型稳定
2. 仓位表达稳定
3. 买入到卖出的语言稳定

## 第一阶段：30 天内先把系统收干净

### 只做三类主路径

- 成长预期差
- 困境 / 政策修复预期差
- 高质量龙头业绩验证

### 这阶段只验收一件事

> 账户里不能再出现“我也不知道为什么买这个”的交易。

### 强制动作

- 每笔交易先写计划卡片
- 不属于主路径的一律放弃
- 所有“别人说的 / 盘中突然想的”直接降级为 C 类

## 第二阶段：30 到 60 天把交易卡片固化

### 每笔交易必须提前写清

- 这笔钱属于什么类型
- 核心变量是什么
- 如何验证
- 当前定价阶段
- A / B / C 等级
- 初始仓位
- 减仓条件
- 清仓条件
- 卖出后买回条件

### 这阶段只验收一件事

> 盘中不再临时找理由，盘后不再用新逻辑替换旧逻辑。

## 第三阶段：60 到 90 天开始看结构数据

### 不先看总收益，先看结构

| 统计项 | 要回答的问题 |
|--------|--------------|
| A 类交易胜率 | 主路径到底有没有优势 |
| A 类平均盈亏比 | 好机会有没有拿住 |
| B 类拖累程度 | B 类值不值得继续做 |
| 最大亏损来自哪里 | 是否在做不属于自己的钱 |
| 情绪型卖出占比 | 卖点有没有被情绪劫持 |
| 最重仓交易质量 | 仓位是否跟质量匹配 |

## 每周动作

1. 清一次当前持仓的赚钱类型
2. 看一次是否有主变量失效
3. 检查有没有被热度和比较效应带偏
4. 更新一次 `Projects/00-当前进度.md`

## 每月动作

1. 更新 `交易复盘目录`
2. 更新 `交易复盘阶段性总结`
3. 统计 A / B / C 类交易结果
4. 重看一次回避列表，确认是否越界

## 未来 6 个月直接回避

- 纯低估值慢修复
- 高股息类债配置
- 拥挤成长后半段大仓位趋势
- 纯消息、纯题材、纯情绪接力

## 当前最重要的四个训练任务

1. 每笔交易先定义赚钱类型
2. 每笔交易只抓一个主变量
3. 仓位和机会质量严格挂钩
4. 接受很多涨得好的钱本来就不属于自己

## 最后压成一句话

> **先把系统做窄、做稳、做干净，再谈稳定盈利。**
