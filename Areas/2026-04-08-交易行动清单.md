# 交易行动清单

## 开仓前

- [ ] 这笔交易属于三类主路径之一
- [ ] 我能一句话说清这笔钱是什么钱
- [ ] 我只写了一个主变量，最多一个辅助变量
- [ ] 我知道未来 1 到 4 个季度怎么验证
- [ ] 我知道市场现在是“没看到 / 刚看到 / 已一致 / 已拥挤”哪一段
- [ ] 我已经给它分了 A / B / C
- [ ] 仓位和 A / B / C 等级匹配
- [ ] 我已经提前写好减仓、清仓、买回条件

## 持有中

- [ ] 核心变量还在继续兑现
- [ ] 最新信息是在强化逻辑，而不是改写逻辑
- [ ] 我没有因为价格上涨就自动加仓
- [ ] 我没有把中期交易临时改成长期信仰
- [ ] 我没有因为别的票涨得更快就怀疑原计划

## 卖出前

- [ ] 这是逻辑失效，不是情绪失控
- [ ] 或者这是节奏走坏，不是盘中恐慌
- [ ] 或者这是兑现完成，不是随手止盈
- [ ] 卖出后如果条件重现，我愿意按规则买回

## 卖出后

- [ ] 我能复盘这笔钱到底属于哪一类
- [ ] 我能说清做对了什么
- [ ] 我能说清做错了什么
- [ ] 我知道它以后是“继续做 / 只小仓位做 / 直接回避”

## 每周复盘

- [ ] 本周新交易是否都属于主路径
- [ ] 是否出现了“听消息就下单”的噪音入口
- [ ] 是否有仓位超过机会质量
- [ ] 是否有卖点被情绪劫持
- [ ] 是否有本不该做但因为热度去做的交易

## 每月复盘

- [ ] A 类交易胜率和盈亏比
- [ ] B 类交易是否在拖累账户
- [ ] 亏损最大的交易是否都偏离主路径
- [ ] 情绪型卖出占比
- [ ] 最重仓的交易，是否也是质量最高的交易

## 直接回避列表

- [ ] 纯低估值慢修复
- [ ] 高股息类债配置
- [ ] 高拥挤成长后半段重仓追涨
- [ ] 纯题材、纯消息、纯情绪接力
- [ ] 自己讲不清、也无法验证的机会

## 日常提醒

> **不是看到机会就做，而是只做属于自己的机会。**
